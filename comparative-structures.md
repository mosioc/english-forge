## 19.2 Comparative Structures

Comparative structures are used to compare differences or similarities between two or more things, people, or ideas. They involve adjectives, adverbs, and specific grammatical patterns.

> **Related Sections:** See *[Section 19. Adjectives and Adverbs](./adjectives-adverbs.md)* for comparative and superlative forms, *[Section 19.1 Order of Adjectives](./order-of-adjectives.md)* for adjective placement, and *[14. Verb Complementation: Gerunds (-ing) and Infinitives (to) (Verbals)](./ing-and-to-forms-verbals.md)* for verb patterns after comparisons.

---

### A. Types of Comparison

| Type | Function | Example |
|------|----------|---------|
| **Comparative** | Compares two things | *She is **taller than** her brother.* |
| **Superlative** | Compares three or more things | *She is **the tallest** in the class.* |
| **Equality** | Shows two things are equal | *He is **as fast as** his friend.* |
| **Inequality** | Shows two things are not equal | *This book is **not as interesting as** that one.* |
| **Proportional** | Shows a relationship between two changes | ***The more** you study, **the better** you do.* |

---

### B. Comparative Adjectives and Adverbs

#### 1. Forming Comparatives

| Adjective/Adverb Type | Rule | Example |
|----------------------|------|---------|
| **One syllable** | Add *-er* | *tall -> taller, fast -> faster, big -> bigger* |
| **One syllable (ending in -e)** | Add *-r* | *nice -> nicer, late -> later, large -> larger* |
| **One syllable (CVC pattern\*)** | Double final consonant + *-er* | *big -> bigger, hot -> hotter, sad -> sadder* |
| **Two syllables ending in -y** | Change *y* to *i* + *-er* | *happy -> happier, easy -> easier, busy -> busier* |
| **Two syllables (other)** | *more* + adjective | *peaceful -> more peaceful, careful -> more careful* |
| **Three or more syllables** | *more* + adjective | *expensive -> more expensive, beautiful -> more beautiful* |
| **Adverbs ending in -ly** | *more* + adverb | *quickly -> more quickly, carefully -> more carefully* |
| **Irregular forms** | Special forms | *good/well -> better, bad/badly -> worse, far -> farther/further* |

> *CVC pattern: a Consonant, then a Vowel, then a Consonant.

#### 2. Using Comparatives

> **Structure:** *Comparative + than*

| Example |
|---------|
| *My house is **bigger than** yours.* |
| *She runs **faster than** her brother.* |
| *This exam is **more difficult than** the last one.* |
| *He speaks English **better than** I do.* |
| *The weather today is **worse than** yesterday.* |

#### 3. Modifying Comparatives (Degree)

Use these words before a comparative to show degree:

| Modifier | Meaning | Example |
|----------|---------|---------|
| *much / far / a lot* | Much more | *She is **much** taller than me.* |
| *a little / a bit / slightly* | Slightly more | *It's **a little** colder today.* |
| *even* | Emphasizes difference | *He is **even** faster than before.* |
| *no* | Equal in negative sense | *I am **no** richer than you.* |
| *any* | Used in questions/negatives | *Are you **any** better today?* |

**Examples:**
- *This car is **far more expensive** than that one.*
- *She is **a bit older** than her husband.*
- *He is **no taller** than his younger brother.*

---

### C. Superlative Adjectives and Adverbs

#### 1. Forming Superlatives

| Adjective/Adverb Type | Rule | Example |
|----------------------|------|---------|
| **One syllable** | Add *-est* | *tall -> the tallest, fast -> the fastest* |
| **One syllable (ending in -e)** | Add *-st* | *nice -> the nicest, large -> the largest* |
| **Two syllables ending in -y** | Change *y* to *i* + *-est* | *happy -> the happiest, easy -> the easiest* |
| **Two syllables (other)** | *the most* + adjective | *peaceful -> the most peaceful* |
| **Three or more syllables** | *the most* + adjective | *expensive -> the most expensive* |
| **Irregular forms** | Special forms | *good/well -> the best, bad/badly -> the worst* |

#### 2. Using Superlatives

> **Structure:** *the + superlative + (noun) + in/of + group*

| Example |
|---------|
| *He is **the tallest** in the class.* |
| *This is **the most beautiful** place **in** the world.* |
| *She runs **the fastest** **of** all the athletes.* |
| *That was **the best** meal I've ever had.* |
| *It was **the worst** day **of** my life.* |

> **Note:** Use *in* before singular places or groups (*in the class, in the world*). Use *of* before plural or before a specific set (*of all, of them*).

---

### D. Equality and Inequality Structures

#### 1. Equality: *as... as*

> **Structure:** *as + adjective/adverb + as*

| Example |
|---------|
| *She is **as tall as** her mother.* |
| *He runs **as fast as** his brother.* |
| *This book is **as interesting as** that one.* |
| *I can't sing **as well as** her.* |

#### 2. Negative Equality: *not as... as* / *not so... as*

> **Structure:** *not as/so + adjective/adverb + as*

| Example |
|---------|
| *This movie is **not as good as** the book.* |
| *He is **not so tall as** his father.* (more formal) |
| *She doesn't sing **as beautifully as** her sister.* |

#### 3. Inequality: *less... than*

> **Structure:** *less + adjective/adverb + than* (opposite of *more... than*)

| Example |
|---------|
| *This problem is **less difficult than** the previous one.* |
| *She speaks **less fluently than** her friend.* |
| *This phone is **less expensive than** that one.* |

---

### E. Proportional Comparisons (The... The...)

These structures show that two things change together. One change causes or corresponds to another change.

> **Structure:** *The + comparative + (subject + verb), the + comparative + (subject + verb)*

| Example | Meaning |
|---------|---------|
| ***The more** you study, **the better** you do.* | Studying more leads to doing better |
| ***The harder** he works, **the more** he earns.* | Hard work leads to more earnings |
| ***The sooner** we leave, **the sooner** we will arrive.* | Earlier departure means earlier arrival |
| ***The more expensive** the hotel, **the better** the service.* | Higher price corresponds to better service |
| ***The less** you worry, **the happier** you will be.* | Less worry leads to more happiness |

**Variations:**
- *The more, the merrier.* (Shortened form)
- *The sooner, the better.*
- *The longer I wait, the more impatient I become.*

---

### F. Repeated Comparatives (Progressive Change)

These structures show continuous increase or decrease.

> **Structure:** *comparative and comparative*

| Example | Meaning |
|---------|---------|
| *The weather is getting **colder and colder**.* | Continuously colder |
| *She became **more and more** beautiful.* | Continuously more beautiful |
| *He ran **faster and faster**.* | Continuously faster |
| *The problem is becoming **more and more** difficult.* | Continuously more difficult |
| *As the day went on, it got **hotter and hotter**.* | Progressive increase in heat |

---

### G. Comparing Quantities

Use these structures to compare amounts or numbers.

| Structure | Example |
|-----------|---------|
| ***more** + noun + **than*** | *She has **more friends than** anyone else.* |
| ***fewer** + plural noun + **than*** | *There are **fewer students than** last year.* |
| ***less** + uncountable noun + **than*** | *I have **less time than** I thought.* |
| ***as much** + uncountable noun + **as*** | *He earns **as much money as** his wife.* |
| ***as many** + plural noun + **as*** | *She invited **as many people as** possible.* |

**Examples:**
- *I have **more** books **than** you.*
- *There is **less** sugar in this recipe **than** in that one.*
- *She has **as much experience as** her manager.*
- *We need **fewer** meetings **than** before.*

---

### H. Comparing Actions (Verb Phrases)

When comparing actions, use *than* with a clause or a reduced clause.

| Structure | Example |
|-----------|---------|
| **Full clause** | *She sings better **than I sing**.* |
| **Reduced clause (auxiliary only)** | *She sings better **than I do**.* |
| **Reduced clause (subject only)** | *She sings better **than me**.* (informal) |
| **Infinitive comparison** | *It's better **to be safe than to be sorry**.* |
| **Gerund comparison** | ***Eating healthy is better than eating junk food**.* |

**Examples:**
- *He runs faster **than she does**.*
- *They arrived earlier **than we expected**.*
- *It's easier **to criticize than to create**.*

---

### I. Special Comparative Patterns

#### 1. *No sooner... than...*

Shows that one event happened immediately after another.

> **Structure:** *No sooner + auxiliary + subject + verb + than + clause*

| Example |
|---------|
| ***No sooner had I left** than it started raining.* |
| ***No sooner did he arrive** than he had to leave again.* |

#### 2. *The same... as...*

Shows equality of a specific quality.

> **Structure:** *the same + (noun) + as*

| Example |
|---------|
| *Your car is **the same color as** mine.* |
| *She has **the same problem as** her brother.* |
| *This costs **the same as** that.* |

#### 3. *Different from / to / than*

| Structure | Region | Example |
|-----------|--------|---------|
| *different **from*** | Standard everywhere | *My opinion is different **from** yours.* |
| *different **to*** | British English | *This is different **to** what I expected.* |
| *different **than*** | American English (informal) | *Things are different **than** they used to be.* |

#### 4. *Like* and *Unlike* (Similarity and Contrast)

> **Structure:** *like/unlike + noun*

| Example |
|---------|
| ***Like** his father, he is a doctor.* |
| ***Unlike** her sister, she loves coffee.* |
| *He looks **like** his brother.* |

### J. Common Errors and Corrections

| Error | Correction | Rule |
|-------|------------|------|
| *This is **more better**.* | *This is **better**.* | Double comparatives are incorrect |
| *She is **more taller** than me.* | *She is **taller** than me.* | Use *-er* or *more*, not both |
| *He is **more fast** than her.* | *He is **faster** than her.* | One-syllable adjectives take *-er* |
| *She is the **most fastest**.* | *She is the **fastest**.* | Double superlatives are incorrect |
| *This is **as better as** that.* | *This is **as good as** that.* | Use *as... as* with positive form |
| *He runs **more faster** than me.* | *He runs **much faster** than me.* | Use *much/far* to modify comparatives |
| *The **more** I study, I learn **more**.* | *The **more** I study, **the more** I learn.* | Use *the + comparative* in both clauses |
| *She is **more intelligent than** her brother **is intelligent**.* | *She is **more intelligent than** her brother.* | Avoid repeating the adjective |

### K. Quick Reference Summary

| Structure | Form | Example |
|-----------|------|---------|
| **Comparative** | *adjective/adverb-er + than* | *taller than, faster than* |
| **Comparative (long)** | *more + adjective/adverb + than* | *more beautiful than* |
| **Superlative** | *the + adjective/adverb-est* | *the tallest, the fastest* |
| **Superlative (long)** | *the most + adjective/adverb* | *the most beautiful* |
| **Equality** | *as + adjective/adverb + as* | *as tall as, as fast as* |
| **Negative equality** | *not as/so + adjective/adverb + as* | *not as tall as* |
| **Inequality** | *less + adjective/adverb + than* | *less difficult than* |
| **Proportional** | *the + comparative..., the + comparative...* | *the more, the better* |
| **Repeated comparative** | *comparative and comparative* | *colder and colder* |
| **Quantity comparison** | *more/fewer/less... than* | *more books than* |
| **Equality quantity** | *as much/many... as* | *as much money as* |
| **Same as** | *the same + noun + as* | *the same color as* |
