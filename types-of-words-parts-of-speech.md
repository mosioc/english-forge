## 1.8 Types of Words (Parts of Speech)

Every English word belongs to one of eight (or nine) categories based on its function in a sentence.

| Part of Speech | Function | Examples |
|----------------|----------|----------|
| **Noun** | Names a person, place, thing, or idea. | *dog, city, happiness, Maria* |
| **Pronoun** | Replaces a noun. | *he, she, it, they, who, which* |
| **Verb** | Expresses action or state of being. | *run, think, is, become* |
| **Adjective** | Describes a noun or pronoun. | *big, happy, red, beautiful* |
| **Adverb** | Modifies a verb, adjective, or another adverb. | *quickly, very, often, well* |
| **Preposition** | Shows relationship between a noun/pronoun and another word. | *in, on, at, under, before, with* |
| **Conjunction** | Connects words, phrases, or clauses. | *and, but, because, although* |
| **Article / Determiner** | Specifies a noun. | *a, an, the, this, my, some* |
| **Interjection** | Expresses emotion or reaction. | *Oh!, Wow!, Ouch!, Hey!* |

### 1. Nouns
Person, place, thing, or idea. *See [Section 15. Articles and Nouns](./articles-nouns.md)*

| Type | Definition | Examples |
|------|------------|----------|
| **Concrete** | Physical things | *table, rain, dog* |
| **Abstract** | Concepts or qualities | *love, freedom, happiness* |
| **Countable** | Can be counted | *apple, chairs, ideas* |
| **Uncountable** | Cannot be counted individually | *water, information, advice* |
| **Proper** | Specific names | *Maria, London, English* |

### 2. Pronouns
Replace a noun. *See [Section 16. Pronouns and Determiners](./pronouns-determiners.md)*

| Type | Examples |
|------|----------|
| **Subject** | *I, you, he, she, it, we, they* |
| **Object** | *me, you, him, her, it, us, them* |
| **Possessive** | *mine, yours, his, hers, ours, theirs* |
| **Reflexive** | *myself, yourself, himself, herself, itself, ourselves, themselves* |
| **Demonstrative** | *this, that, these, those* |
| **Interrogative** | *who, whom, whose, which, what* |
| **Relative** | *who, whom, whose, which, that* |
| **Indefinite** | *someone, anything, no one, everything* |
| **Reciprocal** | *each other, one another* |

### 3. Verbs
Express action or state of being. *See [Section 1.7 Types of Verbs](./types-of-verbs.md)*

| Type | Function | Examples |
|------|----------|----------|
| **Action** | Physical or mental action | *run, eat, think, write* |
| **Linking** | Connects subject to description | *is, seem, become, look* |
| **Auxiliary** | Helps main verb form tense/voice | *be, have, do* |
| **Modal** | Expresses possibility/obligation | *can, must, should, will* |

### 4. Adjectives
Describe a noun or pronoun. *See [Section 19. Adjectives and Adverbs](./adjectives-adverbs.md)*

| Type | Function | Examples |
|------|----------|----------|
| **Descriptive** | Describe quality | *big, beautiful, red* |
| **Comparative** | Compare two | *bigger, more beautiful* |
| **Superlative** | Compare three or more | *biggest, most beautiful* |
| **Demonstrative** | Point to specific noun | *this, that, these, those* |
| **Quantitative** | Show quantity | *some, many, few, five* |
| **Possessive** | Show ownership | *my, your, his, her, our, their* |
| **Interrogative** | Ask questions | *which, what, whose* |

### 5. Adverbs
Modify verbs, adjectives, or other adverbs. *See [Section 19. Adjectives and Adverbs > Adverbs](./adjectives-adverbs.md#b-adverbs)*

| Type | Answers | Examples |
|------|---------|----------|
| **Manner** | How? | *quickly, carefully, well* |
| **Frequency** | How often? | *always, often, never, sometimes* |
| **Time** | When? | *now, yesterday, soon, later* |
| **Place** | Where? | *here, there, outside, inside* |
| **Degree** | To what extent? | *very, too, enough, extremely* |
| **Certainty** | How sure? | *probably, definitely, maybe* |
| **Viewpoint** | From what perspective? | *personally, technically, honestly* |
| **Interrogative** | Ask questions | *when, where, why, how* |
| **Relative** | Connect clauses | *when, where, why* |

### 6. Prepositions
Show relationship between a noun/pronoun and another word. *See [Section 21. Prepositions](./prepositions.md)*

| Type | Examples |
|------|----------|
| **Time** | *at, on, in, before, after, during, since, for, until, by* |
| **Place** | *at, on, in, under, over, behind, between, among, near* |
| **Direction** | *to, from, into, onto, towards, through, across* |
| **Other** | *with, without, about, of, for, by, like, as, except* |

> **Key Rule:** Verbs after prepositions take the *-ing* form. *interested **in learning**, good **at singing**.*

### 7. Conjunctions
Connect words, phrases, or clauses. *See [Section 20. Conjunctions](./conjunctions.md)*

| Type | Function | Examples |
|------|----------|----------|
| **Coordinating** (FANBOYS) | Connect equal elements | *for, and, nor, but, or, yet, so* |
| **Subordinating** | Introduce dependent clauses | *because, although, if, when, while, unless, since* |
| **Correlative** | Paired conjunctions | *both... and, either... or, neither... nor, not only... but also* |

### A. Key Distinctions and Common Errors 

| Often Confused | Difference | Examples |
|----------------|------------|----------|
| **Adjective vs. Adverb** | Adjective describes a noun; adverb describes a verb/adjective/adverb. | *She is **careful**.* (adj) / *She drives **carefully**.* (adv) |
| **Preposition vs. Conjunction** | Preposition + noun; conjunction + clause. | *I left **after** the game.* (prep) / *I left **after** the game ended.* (conj) |
| **Determiner vs. Pronoun** | Determiner comes before a noun; pronoun replaces a noun. | *This book is mine.* (det) / *This is mine.* (pron) |

### B. Quick Reference: Questions They Answer

| Part of Speech | Question It Answers |
|----------------|---------------------|
| **Noun** | Who? What? |
| **Verb** | What action? What state? |
| **Adjective** | What kind? Which one? How many? |
| **Adverb** | How? When? Where? Why? To what degree? |
| **Preposition** | Where? When? How? (relationship) |
