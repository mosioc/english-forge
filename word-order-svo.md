## 1.5 Word Order (SVO)

English is an **SVO language**: the core word order is **Subject + Verb + Object**. Understanding this basic structure and the rules for placing other elements (indirect objects, adverbs, prepositional phrases) is essential for forming correct sentences.

> **Related Sections:** See *[Section 19. Adjectives and Adverbs](./adjectives-adverbs.md)* for adverb placement with verbs and adjectives, *[Section 11. Inversion](./inversion.md)* for when normal word order is reversed, *[Section 13. Questions](./questions.md)* for question word order, and *[Section 19.1 Order of Adjectives](./order-of-adjectives.md)* for order of adjectives.

### A. Basic Sentence Structure: Subject + Verb + Object

The standard English sentence follows this pattern:

| Element | Description | Example |
|---------|-------------|---------|
| **Subject (S)** | Who or what performs the action | ***She*** *reads.* |
| **Verb (V)** | The action or state | *She* ***reads*** *.* |
| **Object (O)** | Who or what receives the action | *She reads* ***a book*** *.* |

**Core Patterns:**

| Pattern | Structure | Example |
|---------|-----------|---------|
| **Intransitive** (no object) | S + V | *The baby **cried**.* |
| **Transitive** (direct object) | S + V + DO | *She **bought a car**.* |
| **Ditransitive** (two objects) | S + V + IO + DO | *He **gave me a gift**.* |
| **Linking** (subject complement) | S + V + SC | *She **is happy**.* / *He **became a doctor**.* |
| **Object complement** | S + V + O + OC | *They **painted the room blue**.* |

### B. Direct and Indirect Objects

When a verb takes two objects, English has **two possible word orders**.

#### 1. Verb + Indirect Object + Direct Object (V + IO + DO)

This is the more common pattern in everyday English. No preposition is used.

| Structure | Example |
|-----------|---------|
| S + V + IO + DO | *He gave **me** (IO) **a book** (DO).* |
| | *She sent **her friend** (IO) **a letter** (DO).* |
| | *I'll make **you** (IO) **some coffee** (DO).* |
| | *They offered **the guests** (IO) **a drink** (DO).* |

**Common verbs that follow this pattern:**

| Verb | Example |
|------|---------|
| *give* | *Give **me** a chance.* |
| *send* | *Send **us** the details.* |
| *tell* | *Tell **him** the truth.* |
| *show* | *Show **me** the photos.* |
| *offer* | *Offer **her** a seat.* |
| *lend* | *Lend **me** your pen.* |
| *make* | *Make **me** a promise.* |
| *buy* | *Buy **yourself** a treat.* |

#### 2. Verb + Direct Object + *to/for* + Indirect Object (V + DO + to/for + IO)

This pattern places the direct object first and uses the preposition *to* or *for* before the indirect object.

| Preposition | Example |
|-------------|---------|
| **to** (direction of action) | *He gave a book **to me**.* |
| | *She sent a letter **to her friend**.* |
| | *I explained the problem **to my boss**.* |
| **for** (benefit) | *I'll make some coffee **for you**.* |
| | *She bought a gift **for her mother**.* |
| | *Can you carry this bag **for me**?* |

**When to use the *to/for* pattern:**

| Reason | Example |
|--------|---------|
| The indirect object is a long phrase | *I gave the documents **to the woman who called earlier**.* (better than: *I gave the woman who called earlier the documents.*) |
| You want to emphasize the indirect object | *She gave the award **to him**, not to his brother.* |
| With certain verbs that never use V+IO+DO | *The teacher explained the lesson **to the students**.* (Not: *explained the students the lesson*) |
| The direct object is a pronoun | *Give it **to me**.* (Not: *Give me it.* — though some dialects accept this) |

**Verbs that require *to* (do not allow V+IO+DO):**

| Verb | Correct | Incorrect |
|------|---------|-----------|
| *explain* | *Explain the problem **to me**.* | *Explain me the problem.* |
| *describe* | *Describe the scene **to us**.* | *Describe us the scene.* |
| *suggest* | *He suggested a solution **to me**.* | *He suggested me a solution.* |
| *recommend* | *I recommended the film **to them**.* | *I recommended them the film.* |
| *say* | *He said nothing **to her**.* | *He said her nothing.* |

#### 3. Direct Object as Pronoun

When both objects are pronouns, the preposition pattern is preferred.

| Pattern | Example |
|---------|---------|
| V + DO (pronoun) + *to/for* + IO (pronoun) | *Give **it to me**.* |
| | *Send **them to her**.* |
| | *I'll make **it for you**.* |

> **Note:** When the direct object is not a pronoun, both patterns are usually possible:
> - *Give me the book.* = *Give the book to me.*
> - But: *Give it to me.* (preferred) / *Give me it.* (marginally acceptable)

### C. Place and Time Expressions: The "Place Before Time" Rule

When a sentence includes both a **place** expression and a **time** expression at the end, English follows a fixed order:

> **Rule:** *Place (Where?) comes before Time (When?)*

#### 1. The Basic Pattern

| Correct (Place -> Time) | Incorrect (Time -> Place) |
|-------------------------|---------------------------|
| *She went **to the park** (place) **yesterday** (time).* | *She went yesterday to the park.* |
| *I'll meet you **at the station** (place) **at 5 p.m.** (time).* | *I'll meet you at 5 p.m. at the station.* |
| *They arrived **at the hotel** (place) **in the evening** (time).* | *They arrived in the evening at the hotel.* |
| *We studied **in the library** (place) **all afternoon** (time).* | *We studied all afternoon in the library.* |

#### 2. Expanded Pattern: Manner + Place + Time

When manner (how?) is also included:

> **Rule:** *Manner (How?) -> Place (Where?) -> Time (When?)*

| Pattern | Example |
|---------|---------|
| M -> P -> T | *She drove **carefully** (manner) **to work** (place) **this morning** (time).* |
| M -> P -> T | *They waited **patiently** (manner) **in the queue** (place) **for hours** (time).* |
| M -> P -> T | *He walked **slowly** (manner) **to the door** (place) **at midnight** (time).* |

#### 3. Exceptions: When Time Comes Before Place

Time can come first when it is **the main focus**, especially in shorter sentences.

| Standard (P -> T) | Time-First (T -> P) | When to use T -> P |
|-------------------|---------------------|---------------------|
| *I was in London **in June**.* | *I was in London in June.* | (same length — both fine) |
| *She was born **in this hospital** **in 1990**.* | ***In 1990**, she was born in this hospital.* | Emphasizing the year |
| *We went to the beach **last summer**.* | ***Last summer** we went to the beach.* | Short time phrase — more natural |

**Guidelines for Time-First order:**

| Condition | Example |
|-----------|---------|
| Time is short and introduces the topic | ***Yesterday** I met an old friend.* |
| Time contrasts with another time | ***Last year** we went to France; **this year** we're going to Italy.* |
| Place is a long phrase | ***At 8 a.m.** we arrived at the small hotel on top of the hill.* |

> **Avoid this common error:** Don't separate the verb from its object with a time or place expression.
> - ✓ *I read **the book** yesterday.*
> - ✗ *I read yesterday **the book**.*

### D. Adverb Placement: Quick Summary

For full details, see *[Section 19. Adjectives and Adverbs](./adjectives-adverbs.md)*. Here is a short summary:

| Adverb Type | Typical Position | Example |
|-------------|------------------|---------|
| **Frequency** (*always, often, never*) | Before main verb / After *be* | *She **always** arrives early.* / *He **is never** late.* |
| **Manner** (*carefully, quickly*) | After verb + object | *She opened the box **carefully**.* |
| **Place** (*here, outside*) | End position | *Let's go **outside**.* |
| **Time** (*yesterday, tomorrow*) | End position (or front for emphasis) | *I'll call you **tomorrow**.* / ***Tomorrow** I'll call you.* |
| **Certainty** (*probably, definitely*) | Mid position (before main verb, after auxiliary) | *She **will definitely** come.* |
| **Viewpoint** (*personally, technically*) | Beginning of clause | ***Personally**, I disagree.* |

### E. Common Word Order Errors to Avoid

| Error | Correction | Rule |
|-------|------------|------|
| *She went yesterday to the park.* | *She went to the park **yesterday**.* | Place before Time |
| *I'll meet at 5 p.m. at the station you.* | *I'll meet **you** at the station at 5 p.m.* | Object comes immediately after verb |
| *He told to me a story.* | *He told **me** a story.* | V + IO + DO has no preposition |
| *Can you explain me the problem?* | *Can you explain the problem **to me**?* | *Explain* requires *to* with indirect object |
| *I read yesterday a good book.* | *I read a good book **yesterday**.* | Don't separate verb from its object |
| *She drives to work every day carefully.* | *She drives **carefully** to work every day.* | Manner -> Place -> Time |
| *They arrived in the evening at the airport.* | *They arrived **at the airport** in the evening.* | Place before Time |
| *She gave it to me.* (correct) | *She gave me it.* (marginally acceptable) | Pronoun objects -> use *to/for* pattern |
| *I bought for her a present.* | *I bought **her** a present.* / *I bought a present **for her**.* | Follow V + IO + DO or V + DO + *for* + IO |
| *He suggested me a good restaurant.* | *He suggested a good restaurant **to me**.* | *Suggest* requires *to*, not V+IO+DO |

### F. Quick Reference Summary Table

| What to Place | Correct Order | Example |
|---------------|---------------|---------|
| **Core structure** | S + V + O | *She reads a book.* |
| **Two objects (pattern 1)** | S + V + IO + DO | *He gave me a gift.* |
| **Two objects (pattern 2)** | S + V + DO + *to/for* + IO | *He gave a gift to me.* |
| **After the object** | Manner -> Place -> Time | *She waited patiently at the station for an hour.* |
| **Front position (emphasis)** | Time -> S + V | *Yesterday she waited at the station.* |
| **Adverb (frequency)** | Before main verb / After *be* | *She often visits.* / *He is always late.* |
