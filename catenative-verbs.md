# 14.2 Catenative Verbs

## A. What Are Catenative Verbs?

A **catenative verb** (from Latin *catena* = "chain") is a verb that can be followed directly by another verb in a chain. The second verb appears in one of four forms:

| Form | Example |
|------|---------|
| **Bare Infinitive** (base verb without *to*) | *She let him **go**.* |
| **To-Infinitive** | *He wants **to leave**.* |
| **Gerund (-ing)** | *She enjoys **swimming**.* |
| **Either (same meaning)** | *I started **to cry** / **crying**.* |
| **Either (different meaning)** | *I stopped **to smoke** / **smoking**.* |

---

## B. Pattern 1: Verb + Bare Infinitive

Used with a small set of verbs, mostly modals and causatives.

### 1. Modal Auxiliary Verbs

All modal verbs are followed by the bare infinitive.

| Modal | Example |
|-------|---------|
| *can* | *She **can speak** French.* |
| *could* | *He **could swim** when he was five.* |
| *will* | *They **will arrive** tomorrow.* |
| *would* | *I **would go** if I had time.* |
| *shall* | *We **shall overcome**.* |
| *should* | *You **should rest**.* |
| *may* | *She **may come** later.* |
| *might* | *It **might rain**.* |
| *must* | *You **must leave** now.* |

> **See also:** *[Section 6. Modal Verbs](./modal-verbs.md)*

### 2. Causative Verbs

| Verb | Structure | Example | Meaning |
|------|-----------|---------|---------|
| *make* | make + object + bare infinitive | *She **made him cry**.* | Force |
| *have* | have + object + bare infinitive | *I **had the plumber fix** the sink.* | Arrange/Instruct |
| *let* | let + object + bare infinitive | *They **let us go** early.* | Allow |
| *help* | help + object + bare infinitive / to-infinitive | *She **helped me (to) move**.* | Assist |

> **Important:** *Get* is also causative but takes *to*-infinitive: *I got him **to help**.*

### 3. Verbs of Perception

Verbs of perception can be followed by bare infinitive *or* -ing form (with a meaning difference).

| Verb | Bare Infinitive | -ing Form |
|------|-----------------|-----------|
| *see* | *I saw him **cross** the road.* (complete action) | *I saw him **crossing** the road.* (action in progress) |
| *hear* | *She heard him **slam** the door.* | *She heard him **slamming** the door.* |
| *watch* | *We watched them **play**.* | *We watched them **playing**.* |
| *feel* | *I felt the ground **shake**.* | *I felt the ground **shaking**.* |
| *notice* | *I noticed him **leave**.* | *I noticed him **leaving**.* |

### 4. *Had Better* and *Would Rather*

| Expression | Example |
|------------|---------|
| *had better* | *You **had better leave** now.* |
| *would rather* | *I **would rather stay** home.* |

## C. Pattern 2: Verb + To-Infinitive

This is the most common catenative pattern.

### Common Verbs Followed by To-Infinitive

| Category | Verbs |
|----------|-------|
| **Desire / Intention** | *want, wish, hope, intend, plan, mean, aim, desire, long, yearn* |
| **Decision / Effort** | *decide, choose, agree, refuse, promise, offer, attempt, try, manage, fail, struggle, strive* |
| **Expectation** | *expect, hope, anticipate* |
| **Mental Process** | *learn, forget, remember, pretend, claim* |
| **Appearance** | *appear, seem, prove, turn out* |
| **Obligation / Necessity** | *need, have, ought* |
| **Causative (with *get*)** | *get* (get + object + *to*-infinitive) |

| Example |
|---------|
| *I **want to go**.* |
| *She **decided to stay**.* |
| *They **agreed to help**.* |
| *He **appears to be** tired.* |
| *We **expect to arrive** by noon.* |
| *You **need to rest**.* |
| *I **got him to agree**.* |

### Verb + Object + To-Infinitive

Many verbs can take an object between the main verb and the infinitive.

| Verb Pattern | Example |
|--------------|---------|
| *want + object + to-infinitive* | *I want **you to come**.* |
| *ask + object + to-infinitive* | *She asked **me to help**.* |
| *tell + object + to-infinitive* | *He told **us to wait**.* |
| *advise + object + to-infinitive* | *I advised **her to rest**.* |
| *encourage + object + to-infinitive* | *They encouraged **him to apply**.* |
| *force + object + to-infinitive* | *She forced **me to confess**.* |
| *invite + object + to-infinitive* | *We invited **them to join**.* |
| *persuade + object + to-infinitive* | *I persuaded **her to stay**.* |
| *remind + object + to-infinitive* | *He reminded **me to call**.* |
| *teach + object + to-infinitive* | *She taught **him to swim**.* |
| *warn + object + to-infinitive* | *They warned **us not to go**.* |
| *allow + object + to-infinitive* | *She allowed **me to leave**.* |
| *expect + object + to-infinitive* | *I expect **you to be** on time.* |
| *would like + object + to-infinitive* | *I'd like **you to meet** someone.* |
| *would prefer + object + to-infinitive* | *I'd prefer **you to stay**.* |

## D. Pattern 3: Verb + Gerund (-ing)

### Common Verbs Followed by Gerund

| Category | Verbs |
|----------|-------|
| **Enjoyment / Dislike** | *enjoy, love, like, adore, dislike, hate, detest, loathe, can't stand, can't bear* |
| **Avoidance / Delay** | *avoid, postpone, delay, put off, defer* |
| **Completion / Cessation** | *finish, stop, give up, quit, cease* |
| **Mental Process** | *consider, imagine, fancy, suggest, recommend, mind, miss, recall, anticipate* |
| **Confession / Denial** | *admit, deny* |
| **Practice / Involvement** | *practise, involve, keep, risk* |

| Example |
|---------|
| *She **enjoys dancing**.* |
| *He **avoids speaking** in public.* |
| *They **finished eating**.* |
| *I **miss seeing** you.* |
| *She **admitted stealing** the money.* |
| *He **keeps interrupting** me.* |
| *We **postponed leaving** until Monday.* |

### Verb + Preposition + Gerund

Many verbs require a preposition before the gerund.

| Verb + Preposition | Example |
|--------------------|---------|
| *apologize for* | *He **apologized for being** late.* |
| *insist on* | *She **insisted on paying**.* |
| *succeed in* | *They **succeeded in winning**.* |
| *think of/about* | *I'm **thinking of changing** jobs.* |
| *look forward to* | *We **look forward to seeing** you.* |
| *object to* | *He **objects to working** late.* |
| *accuse (someone) of* | *They **accused him of lying**.* |
| *blame (someone) for* | *She **blamed me for losing** it.* |
| *congratulate (someone) on* | *I **congratulated her on winning**.* |
| *prevent (someone) from* | *The rain **prevented us from going**.* |
| *thank (someone) for* | *He **thanked me for helping**.* |
| *suspect (someone) of* | *They **suspect him of cheating**.* |
| *warn (someone) against* | *I **warned her against going** alone.* |

## E. Pattern 4: Verb + Either Gerund or To-Infinitive (Same Meaning)

These verbs accept both forms with **no significant meaning change**.

| Verb | Gerund Example | Infinitive Example |
|------|----------------|---------------------|
| *begin* | *It **began raining**.* | *It **began to rain**.* |
| *start* | *She **started crying**.* | *She **started to cry**.* |
| *continue* | *He **continued talking**.* | *He **continued to talk**.* |
| *cease* | *They **ceased fighting**.* | *They **ceased to fight**.* |
| *can't bear* | *I **can't bear watching**.* | *I **can't bear to watch**.* |
| *can't stand* | *She **can't stand waiting**.* | *She **can't stand to wait**.* |
| *love* | *I **love swimming**.* | *I **love to swim**.* |
| *hate* | *He **hates waking** early.* | *He **hates to wake** early.* |
| *like* | *She **likes cooking**.* | *She **likes to cook**.* |
| *prefer* | *I **prefer walking**.* | *I **prefer to walk**.* |
| *propose* | *He **proposed leaving**.* | *He **proposed to leave**.* |
| *bother* | *Don't **bother calling**.* | *Don't **bother to call**.* |

**Note:** Some speakers perceive a very slight difference with *like/love/hate/prefer*:
- Gerund = enjoyment of the activity in general
- Infinitive = choice or habit in a specific situation

- *I like swimming.* (I enjoy it)
- *I like to swim before breakfast.* (It's my habit/preference)

## F. Pattern 5: Verb + Either Gerund or To-Infinitive (Different Meaning)

These verbs change meaning depending on the form that follows. **This is one of the most common advanced learner error areas.**

### 1. *Remember* / *Forget* / *Regret*

| Form | Meaning | Example |
|------|---------|---------|
| **Gerund** | Action happens **before** remembering/forgetting/regretting (looking back) | *I **remember locking** the door.* (I locked it, and now I recall that) |
| **To-Infinitive** | Action happens **after** remembering/forgetting/regretting (looking forward) | *I **remembered to lock** the door.* (I didn't forget to do it) |

| Verb | Gerund (Earlier Action) | Infinitive (Later Action) |
|------|-------------------------|---------------------------|
| *remember* | *I remember **visiting** Paris.* (Past memory) | *Remember **to call** your mother.* (Don't forget) |
| *forget* | *I'll never forget **meeting** her.* (Past event) | *I forgot **to pay** the bill.* (Failed to do it) |
| *regret* | *I regret **saying** that.* (I wish I hadn't said it) | *I regret **to inform** you...* (I'm sorry to have to tell you now) |

### 2. *Stop*

| Form | Meaning | Example |
|------|---------|---------|
| **Gerund** | Cease an activity (stop the action) | *He **stopped smoking**.* (He quit) |
| **To-Infinitive** | Pause in order to do something (stop for a purpose) | *He **stopped to smoke**.* (He paused his walk to have a cigarette) |

### 3. *Try*

| Form | Meaning | Example |
|------|---------|---------|
| **Gerund** | Experiment (test a method to see what happens) | *Try **restarting** the computer.* (Do this and see if it helps) |
| **To-Infinitive** | Attempt (make an effort, often difficult) | *I tried **to open** the window but it was stuck.* (I attempted and failed) |

### 4. *Mean*

| Form | Meaning | Example |
|------|---------|---------|
| **Gerund** | Involve, result in, entail | *Getting fit **means exercising** regularly.* |
| **To-Infinitive** | Intend | *I **meant to call** you but forgot.* |

### 5. *Go on*

| Form | Meaning | Example |
|------|---------|---------|
| **Gerund** | Continue an activity | *She **went on talking** for hours.* |
| **To-Infinitive** | Move to a new stage/activity | *After university, he **went on to become** a lawyer.* |

### 6. *Need / Want / Require / Deserve*

| Form | Meaning | Example |
|------|---------|---------|
| **Gerund** | Passive meaning (something needs to be done TO the subject) | *The car **needs washing**.* (= needs to be washed) |
| **To-Infinitive** | Active meaning (subject needs to do something) | *I **need to wash** the car.* |

| Example | Meaning |
|---------|---------|
| *Your hair **needs cutting**.* | = needs to be cut (passive) |
| *The house **wants painting**.* | = needs to be painted (passive) |
| *He **deserves praising**.* | = deserves to be praised (passive) |

## G. Catenative Verb Chains (Multiple Verbs)

Catenative verbs can form long chains.

| Chain Length | Example |
|--------------|---------|
| Two verbs | *I **want to leave**.* |
| Three verbs | *She **seems to want to help**.* |
| Four verbs | *He **appears to want to try to change**.* |
| Five verbs | *They **seem to want to keep trying to improve**.* |

**Word Order Rule:** Each verb determines the form of the verb that follows it.

## H. Full Classification Summary

| Pattern | Key Verbs | Example |
|---------|-----------|---------|
| **Verb + Bare Infinitive** | modals, *make, let, have, help*, perception verbs | *Let me go.* |
| **Verb + To-Infinitive** | *want, decide, agree, expect, seem, need* | *I want to leave.* |
| **Verb + Object + To-Infinitive** | *want, ask, tell, advise, force, allow, expect* | *She told me to wait.* |
| **Verb + Gerund** | *enjoy, avoid, finish, admit, mind, suggest* | *I enjoy swimming.* |
| **Verb + Preposition + Gerund** | *insist on, succeed in, apologize for* | *He insisted on paying.* |
| **Either - Same Meaning** | *begin, start, continue, love, hate, like* | *I began to cry / crying.* |
| **Either - Different Meaning** | *remember, forget, regret, stop, try, mean* | *I stopped to smoke / smoking.* |
| **Gerund with Passive Meaning** | *need, want, require, deserve* + -ing | *The car needs washing.* |

---

## I. Quick Reference: Which Form Follows?

```txt
Does the main verb determine a specific form?
├── YES -> Use that form
│ ├── Modals, make, let, perception -> Bare Infinitive
│ ├── want, decide, agree, expect -> To-Infinitive
│ └── enjoy, avoid, finish, admit -> Gerund
│
└── NO (allows both) ->
├── Same meaning? -> begin, start, like, love, hate, prefer
└── Different meaning? -> remember, forget, regret, stop, try, mean, go on
```

## J. Common Errors to Avoid

| Error | Correction | Why |
|-------|------------|-----|
| *I want go.* | *I want **to** go.* | *Want* requires *to*-infinitive |
| *She enjoys to dance.* | *She enjoys **dancing**.* | *Enjoy* requires gerund |
| *Let me to help you.* | *Let me help you.* | *Let* takes bare infinitive |
| *I stopped smoking a cigarette.* | *I stopped **to smoke** a cigarette.* | *Stop + gerund* = quit; *stop + to* = pause for a purpose |
| *Remember locking the door!* | *Remember **to lock** the door!* | For a future task, use infinitive |
| *The car needs to wash.* | *The car needs **washing** / **to be washed**.* | *Need + gerund* has passive meaning |
| *He made me to cry.* | *He made me **cry**.* | *Make* takes bare infinitive |
| *I got him fix it.* | *I got him **to** fix it.* | *Get* (causative) takes *to*-infinitive |
| *She suggested to go.* | *She suggested **going**.* | *Suggest* requires gerund (or *that*-clause) |
| *I look forward to see you.* | *I look forward to **seeing** you.* | *To* here is a preposition, not part of infinitive |
