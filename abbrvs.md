### Common Professional Abbreviations

| Abbreviation | Meaning                         | Use Case                                           |
| ------------ | ------------------------------- | -------------------------------------------------- |
| FYI          | For Your Information            | “FYI, the report is ready.”                        |
| ASAP         | As Soon As Possible             | “Please send the draft ASAP.”                      |
| EOD          | End of Day                      | “I’ll deliver the document by EOD.”                |
| ETA          | Estimated Time of Arrival       | “What’s the ETA for the update?”                   |
| TBD          | To Be Determined                | “Meeting time is TBD.”                             |
| TBA          | To Be Announced                 | “Next week’s topic is TBA.”                        |
| Re:          | Regarding                       | Used in subject lines, e.g. “Re: Project Proposal” |
| cc / bcc     | Carbon Copy / Blind Carbon Copy | In email recipients                                |
| Attn:        | Attention                       | “Attn: Marketing Department”                       |
| Enc. / Att.  | Enclosure / Attachment          | “Att.: Financial Report”                           |

### Semi-Formal / Internal Communication

| Abbreviation | Meaning                         | Use Case                              |
| ------------ | ------------------------------- | ------------------------------------- |
| BTW          | By The Way                      | “BTW, great work on the last task.”   |
| IMO / IMHO   | In My (Humble) Opinion          | “IMO, this design looks cleaner.”     |
| FYR          | For Your Reference              | “FYR, I’ve attached the slides.”      |
| N/A          | Not Applicable                  | “That field is N/A for this project.” |
| TL;DR        | Too Long; Didn’t Read (summary) | “TL;DR: Focus on phase 2 first.”      |
| PFA          | Please Find Attached            | “PFA the revised document.”           |
| RSVP         | Please respond                  | “RSVP by Friday.”                     |
| FYA          | For Your Action                 | “FYA, please approve the budget.”     |
| OOT          | Out of Town                     | “I’ll be OOT this week.”              |
| OOO          | Out of Office                   | “I’ll be OOO until Monday.”           |

### Less Common but Useful

| Abbreviation | Meaning                                     | Use Case                                  |
| ------------ | ------------------------------------------- | ----------------------------------------- |
| NRN          | No Reply Necessary                          | “Just FYI, NRN.”                          |
| HTH          | Hope This Helps                             | “HTH! Let me know if you need more info.” |
| AFAIK        | As Far As I Know                            | “AFAIK, the issue is fixed.”              |
| PR / QBR     | Progress Report / Quarterly Business Review | “Let’s review this in our next QBR.”      |
| BR / KR      | Best Regards / Kind Regards                 | Common closing lines                      |
