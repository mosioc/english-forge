## 4. Spelling Rules

Spelling changes often occur when adding endings to nouns, verbs, adjectives, and adverbs. This section covers the most common patterns.

---

### A. Adding `-s` / `-es` (Nouns: plurals / Verbs: third-person singular)

**General rule:** Add `-s` to most words.

- book -> books
- idea -> ideas
- work -> works (verb)
- enjoy -> enjoys (verb)

**Add `-es` when the word ends in:** `-s`, `-ss`, `-sh`, `-ch`, `-x`

| Ending | Example |
|--------|---------|
| `-s` | bus -> buses |
| `-ss` | miss -> misses |
| `-sh` | wash -> washes |
| `-ch` | match -> matches / search -> searches |
| `-x` | box -> boxes |

**Also add `-es` to these common verbs and nouns:**

- potato -> potatoes
- tomato -> tomatoes
- do -> does
- go -> goes

---

### B. Words Ending in `-y`

#### Rule 1: Consonant + `-y` -> Change `y` to `i` (or `ie`)

| Before | Change | Examples |
|--------|--------|----------|
| `-s` (plural or verb-s) | y -> ie | baby -> babies / story -> stories / country -> countries / try -> tries / apply -> applies |
| `-ed` | y -> i | hurry -> hurried / study -> studied / apply -> applied / try -> tried |
| `-er` / `-est` | y -> i | easy -> easier -> easiest / heavy -> heavier -> heaviest / lucky -> luckier -> luckiest |
| `-ly` (adverb) | y -> i | easy -> easily / heavy -> heavily / temporary -> temporarily |

> **Do NOT change `y` before `-ing`:** hurry -> hurrying / study -> studying / try -> trying / apply -> applying

#### Rule 2: Vowel + `-y` -> Keep the `y`

(Applies to endings: `-ay`, `-ey`, `-oy`, `-uy`)

- play -> plays / played
- monkey -> monkeys
- enjoy -> enjoys / enjoyed

#### Irregular Exceptions

- day -> daily
- pay -> paid
- lay -> laid
- say -> said

---

### C. Verbs Ending in `-ie` -> Change `-ie` to `y` before `-ing`

- die -> dying
- lie -> lying
- tie -> tying

---

### D. Words Ending in `-e`

#### Verbs

| Before | Rule | Examples |
|--------|------|----------|
| `-ing` | **drop the `e`** | hope -> hoping / smile -> smiling / dance -> dancing / confuse -> confusing |
| `-ed` (past tense) | **keep `e`, add `-d`** | hope -> hoped / smile -> smiled / dance -> danced / confuse -> confused |

**Exceptions (keep `e` before `-ing`):**

- be -> being
- see -> seeing
- agree -> agreeing

#### Adjectives and Adverbs

| Before | Rule | Examples |
|--------|------|----------|
| `-er` / `-est` (comparative/superlative) | add `-r` / `-st` | wide -> wider -> widest / late -> later -> latest / large -> larger -> largest |
| `-ly` (adverb) | **keep the `e`** | polite -> politely / extreme -> extremely / absolute -> absolutely |

**Adjectives ending in `-le`** -> adverbs ending in `-ly` become `-ply` / `-bly` / `-ly`:

- simple -> simply
- terrible -> terribly
- reasonable -> reasonably

---

### E. Doubling Consonants

#### One-Syllable Words (Ending in: vowel + consonant)

Double the final consonant before `-ing`, `-ed`, `-er`, `-est`.

| Base | Ending | Example |
|------|--------|---------|
| stop | p -> pp | stopping / stopped |
| plan | n -> nn | planning / planned |
| rub | b -> bb | rubbing / rubbed |
| big | g -> gg | bigger / biggest |
| wet | t -> tt | wetter / wettest |
| thin | n -> nn | thinner / thinnest |

#### Words with More Than One Syllable

**Double the final consonant ONLY if the final syllable is stressed.**

| Stressed (-> double) | Not Stressed (-> do NOT double) |
|---------------------|-------------------------------|
| pre**FER** -> preferring / preferred | **VIS**it -> visiting / visited |
| per**MIT** -> permitting / permitted | de**VEL**op -> developing / developed |
| re**GRET** -> regretting / regretted | **HAP**pen -> happening / happened |
| be**GIN** -> beginning | re**MEM**ber -> remembering / remembered |

#### British vs. American English

**British English:** Always double `l` before `-ing` / `-ed`

- travel -> travelling / travelled
- cancel -> cancelling / cancelled

**American English:**

- travel -> traveling / traveled
- cancel -> canceling / canceled

#### When NOT to Double the Consonant

| Condition | Examples |
|-----------|----------|
| Ending in two consonants | start -> starting / started / help -> helping / helped / long -> longer / longest |
| Two vowels before the consonant | boil -> boiling / boiled / need -> needing / needed / explain -> explaining / explained / quiet -> quieter / quietest / cheap -> cheaper / cheapest |
| Ending in `w` or `y` (not consonants at end of word) | stay -> staying / stayed / grow -> growing / new -> newer / newest |
