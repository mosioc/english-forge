## 19.1 Order of Adjectives

When multiple adjectives are used before a noun, they must follow a specific order. Native English speakers do this naturally, but learners need to learn the sequence. Using adjectives in the wrong order sounds awkward or incorrect.

> **Related Section:** See *[Section 19. Adjectives and Adverbs](./adjectives-adverbs.md)* for basic adjective information.

---

### A. The Basic Order of Adjectives

Adjectives generally appear in the following order before a noun:

| Order | Category | Examples |
|-------|----------|----------|
| **1** | **Opinion** | *beautiful, ugly, nice, terrible, wonderful, horrible* |
| **2** | **Size** | *big, small, tiny, huge, enormous, large* |
| **3** | **Age** | *old, new, young, ancient, modern, antique* |
| **4** | **Shape** | *round, square, flat, long, short, oval* |
| **5** | **Color** | *red, blue, green, black, white, dark* |
| **6** | **Origin** | *French, American, Chinese, Italian, Asian* |
| **7** | **Material** | *wooden, metal, plastic, silk, cotton, glass* |
| **8** | **Purpose** | *dining (table), sleeping (bag), running (shoes)* |

> **Memory Aid:** **OSASCOMP** = **O**pinion, **S**ize, **A**ge, **S**hape, **C**olor, **O**rigin, **M**aterial, **P**urpose

---

### B. Detailed Breakdown with Examples

#### 1. Opinion (What someone thinks)

Adjectives that express a subjective judgment or evaluation.

| Examples |
|----------|
| *beautiful, ugly, nice, terrible, wonderful, horrible, lovely, delicious, interesting, boring, good, bad, strange, comfortable, expensive* |

- *a **beautiful** old house*
- *a **nice** red dress*
- *a **terrible** cold day*

#### 2. Size (How big or small)

Adjectives that describe physical dimensions.

| Examples |
|----------|
| *big, small, large, tiny, huge, enormous, little, massive, gigantic, short, tall, long, wide, narrow* |

- *a **big** round table*
- *a **tiny** wooden box*
- *an **enormous** ancient building*

#### 3. Age (How old)

Adjectives that describe age or time period.

| Examples |
|----------|
| *old, new, young, ancient, modern, antique, recent, contemporary, old-fashioned* |

- *an **old** wooden chair*
- *a **new** red car*
- *a **modern** Italian suit*

#### 4. Shape (What form or figure)

Adjectives that describe the shape or form of an object.

| Examples |
|----------|
| *round, square, flat, long, short, oval, rectangular, circular, triangular, curved* |

- *a **round** glass table*
- *a **square** wooden box*
- *a **long** black coat*

#### 5. Color (What color)

Adjectives that describe color.

| Examples |
|----------|
| *red, blue, green, yellow, black, white, dark, light, bright, pale, brown, purple, orange* |

- *a **black** leather jacket*
- *a **red** Italian car*
- *a **white** cotton shirt*

#### 6. Origin (Where from)

Adjectives that describe the source, nationality, or place of origin.

| Examples |
|----------|
| *French, American, Chinese, Italian, Asian, European, African, Japanese, Mexican* |

- *a **French** wooden table*
- *an **Italian** leather bag*
- ***Asian** silk fabric*

#### 7. Material (What it is made of)

Adjectives that describe the substance or material.

| Examples |
|----------|
| *wooden, metal, plastic, silk, cotton, glass, paper, gold, silver, iron, leather, woolen* |

- *a **wooden** dining table*
- *a **leather** handbag*
- *a **glass** coffee cup*

#### 8. Purpose (What it is for)

Adjectives that describe the function or intended use. Often a gerund (-ing form) or a noun used as an adjective.

| Examples |
|----------|
| *dining (table), sleeping (bag), running (shoes), cooking (oil), washing (machine), tennis (racket)* |

- *a dining **table*** (a table for dining)
- *a sleeping **bag*** (a bag for sleeping)
- *running **shoes*** (shoes for running)

> **Note:** The purpose adjective is often so closely tied to the noun that it forms a compound noun: *coffee cup*, *bread knife*, *wine glass*.

---

### C. Complete Examples

Putting all the categories together:

| Order | Example |
|-------|---------|
| 1-2-3 | *a **beautiful big old** house* |
| 1-2-5 | *a **lovely small red** flower* |
| 1-4-7 | *a **nice round wooden** table* |
| 2-5-6 | *a **big black Italian** car* |
| 3-5-7 | *an **old white cotton** shirt* |
| 1-2-3-4-5-6-7 | *a **beautiful big old round red French wooden** table* |

---

### D. Multiple Opinion Adjectives

When using more than one **opinion** adjective, the more subjective/general opinion comes before the more specific/objective opinion.

| Order | Example |
|-------|---------|
| General -> Specific | *a **lovely beautiful** view* (rare to use two opinions) |
| Better to use a comma or *and* | *a **lovely and beautiful** view* |

> **Tip:** Avoid stacking too many opinion adjectives. Two is usually the maximum.

---

### E. Commas Between Adjectives

#### 1. When to Use a Comma

Use a comma between adjectives that are from the **same category** (usually both opinion adjectives).

| Example | Explanation |
|---------|-------------|
| *She is a **kind, generous** person.* | Both are opinion adjectives |
| *It was a **cold, windy** day.* | Both are opinion/weather adjectives |
| *He is a **tall, handsome** man.* | Both are opinion (handsome) and size (tall) - comma optional |

#### 2. When NOT to Use a Comma

Do **not** use a comma between adjectives from **different categories**.

| Example | Explanation |
|---------|-------------|
| *a **beautiful old** house* | Opinion + Age (no comma) |
| *a **big red** car* | Size + Color (no comma) |
| *a **French wooden** table* | Origin + Material (no comma) |

#### 3. Using *And* Instead of a Comma

You can use *and* instead of a comma between the last two adjectives, especially with opinion adjectives.

| With Comma | With *And* |
|------------|------------|
| *a kind, generous person* | *a kind **and** generous person* |
| *a cold, windy day* | *a cold **and** windy day* |

> **Note:** Do not use *and* between adjectives from different categories: *a beautiful and old house* ❌ (sounds unnatural)

---

### F. Adjectives After the Noun (Predicative Position)

When adjectives come **after** the noun (after a linking verb like *be, seem, look, feel*), the order rules are more flexible. You usually use *and* between the last two adjectives.

| Before Noun (Attributive) | After Noun (Predicative) |
|---------------------------|--------------------------|
| *a **beautiful old** house* | *The house is **beautiful and old**.* |
| *a **big red** car* | *The car is **big and red**.* |
| *a **nice round wooden** table* | *The table is **nice, round, and wooden**.* |

---

### G. Examples of Correct vs. Incorrect Order

| Correct Order | Incorrect Order | Why Incorrect |
|---------------|-----------------|---------------|
| *a **beautiful small** garden* | *a **small beautiful** garden* | Opinion before Size |
| *a **big old** tree* | *an **old big** tree* | Size before Age |
| *a **new red** car* | *a **red new** car* | Age before Color |
| *a **round wooden** table* | *a **wooden round** table* | Shape before Material |
| *an **Italian leather** bag* | *a **leather Italian** bag* | Origin before Material |
| *a **long black** coat* | *a **black long** coat* | Shape before Color |

---

### H. Exceptions and Special Cases

#### 1. Opinion Adjectives That Usually Come First

| Adjective | Example |
|-----------|---------|
| *good, bad, nice, lovely, terrible, wonderful* | *a **good** old friend* (not *an old good friend*) |
| *beautiful, ugly, pretty* | *a **beautiful** red dress* |

#### 2. Size Adjectives: *Big, Small, Large, Tiny*

These almost always come before age, shape, and color.

| Example |
|---------|
| *a **big** old house* |
| *a **small** round table* |
| *a **large** red car* |

#### 3. Age Adjectives: *Old, New, Young, Ancient*

These come after size but before shape and color.

| Example |
|---------|
| *a big **old** house* |
| *a small **new** car* |
| *an ancient **round** table* |

#### 4. Color Adjectives from Nouns

Some color adjectives are derived from nouns (e.g., *snow-white, sky-blue, blood-red*) and usually come after other colors or stand alone.

| Example |
|---------|
| *a **snow-white** dress* |
| *a **sky-blue** silk scarf* |

#### 5. Numbers and Quantifiers

Numbers usually come before all other adjectives.

| Order | Example |
|-------|---------|
| Number + Opinion + Size + Noun | ***three** beautiful big houses* |
| Number + Color + Noun | ***two** red cars* |

---

### I. Common Errors to Avoid

| Error | Correction |
|-------|------------|
| *a **red new** car* | *a **new red** car* ✓ |
| *a **wooden round** table* | *a **round wooden** table* ✓ |
| *an **old big** house* | *a **big old** house* ✓ |
| *a **leather Italian** bag* | *an **Italian leather** bag* ✓ |
| *a **beautiful and old** house* (before noun) | *a **beautiful old** house* ✓ (no *and*) |
| *a **small beautiful** garden* | *a **beautiful small** garden* ✓ |
