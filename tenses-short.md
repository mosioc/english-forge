# English Tenses (Short)

A **super short cheat-sheet** of **all tenses with formulas** in **Normal, Question, Passive, and Passive Question**, one line per tense for fast memorization:

---

## **PRESENT TENSES**

* **Present Simple:**
  `S + V(s/es)` | `Do/Does + S + V?` | `O + is/are + V3` | `Is/Are + O + V3?`

* **Present Continuous:**
  `S + am/is/are + V-ing` | `Am/Is/Are + S + V-ing?` | `O + is/are + being + V3` | `Is/Are + O + being + V3?`

* **Present Perfect:**
  `S + have/has + V3` | `Have/Has + S + V3?` | `O + has/have + been + V3` | `Has/Have + O + been + V3?`

* **Present Perfect Continuous:**
  `S + have/has + been + V-ing` | `Have/Has + S + been + V-ing?` | — | —

---

## **PAST TENSES**

* **Past Simple:**
  `S + V2` | `Did + S + V?` | `O + was/were + V3` | `Was/Were + O + V3?`

* **Past Continuous:**
  `S + was/were + V-ing` | `Was/Were + S + V-ing?` | `O + was/were + being + V3` | `Was/Were + O + being + V3?`

* **Past Perfect:**
  `S + had + V3` | `Had + S + V3?` | `O + had + been + V3` | `Had + O + been + V3?`

* **Past Perfect Continuous:**
  `S + had + been + V-ing` | `Had + S + been + V-ing?` | — | —

---

## **FUTURE TENSES**

* **Future Simple:**
  `S + will + V` | `Will + S + V?` | `O + will + be + V3` | `Will + O + be + V3?`

* **Future Continuous:**
  `S + will + be + V-ing` | `Will + S + be + V-ing?` | — | —

* **Future Perfect:**
  `S + will + have + V3` | `Will + S + have + V3?` | `O + will + have + been + V3` | `Will + O + have + been + V3?`

* **Future Perfect Continuous:**
  `S + will + have + been + V-ing` | `Will + S + have + been + V-ing?` | — | —

---

**Legend / Shortcuts**

* `S` = Subject, `O` = Object
* `V` = Base verb
* `V2` = Past verb
* `V3` = Past participle
* `V-ing` = Present participle
