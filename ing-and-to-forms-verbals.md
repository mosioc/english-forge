## 14. Verb Complementation: Gerunds (-ing) and Infinitives (to) (Verbals)

The infinitive (`to + base verb`) and the gerund (`-ing` form of the verb) are used in various grammatical constructions. Understanding when to use each - especially after certain verbs, prepositions, and adjectives - is crucial for accurate and natural English.

> **Quick Reference:**
> - **Gerund (`-ing`):** Functions as a noun (e.g., *Swimming is fun.*)
> - **Infinitive (`to + V1`):** Expresses purpose or follows specific verbs (e.g., *I want to swim.*)

---

### A. Verb + `-ing` (Gerunds)

The `-ing` form of a verb, when used as a noun, is called a **gerund**. It functions as a subject, object, or complement.

| Function | Example |
|----------|---------|
| Subject | ***Swimming** is good exercise.* |
| Object | *I enjoy **reading**.* |
| Complement | *His hobby is **collecting stamps**.* |

#### Verbs Commonly Followed by Gerunds

| Verb | Example |
|------|---------|
| *enjoy* | *I enjoy **listening** to music.* |
| *avoid* | *Please avoid **making** noise.* |
| *mind* | *Do you mind **opening** the window?* |
| *finish* | *Have you finished **writing** the report?* |
| *suggest* | *She suggested **going** to the cinema.* |
| *recommend* | *I recommend **trying** the local specialty.* |
| *admit* | *He admitted **stealing** the car.* |
| *deny* | *They denied **knowing** anything about it.* |
| *imagine* | *Can you imagine **living** without the internet?* |
| *consider* | *We are considering **moving** to a new city.* |
| *practice* | *You need to practice **speaking** English every day.* |
| *risk* | *He risked **losing** all his savings.* |
| *postpone / put off* | *They postponed **deciding** until next week.* |
| *quit / give up* | *He quit **smoking** last year.* |
| *miss* | *I miss **talking** to you every day.* |
| *keep (on)* | *Keep **trying**, and you will succeed.* |
| *allow / permit* | *They don't allow **smoking** here.* |

---

### B. Verb + `to` (Infinitives)

The infinitive form (`to + base verb`) expresses purpose or follows specific verbs.

| Function | Example |
|----------|---------|
| Purpose | *I went to the store **to buy** some milk.* |
| Subject Complement | *His dream is **to travel** the world.* |

#### Verbs Commonly Followed by Infinitives

| Verb | Example |
|------|---------|
| *want* | *I want **to learn** a new language.* |
| *need* | *You need **to study** for the exam.* |
| *hope* | *We hope **to see** you soon.* |
| *plan* | *They plan **to visit** Japan next year.* |
| *decide* | *She decided **to accept** the offer.* |
| *agree* | *He agreed **to help** us.* |
| *offer* | *The company offered **to pay** for the damages.* |
| *promise* | *I promise **to be** there on time.* |
| *fail* | *He failed **to impress** the judges.* |
| *learn* | *She is learning **to drive**.* |
| *manage* | *They managed **to finish** the project.* |
| *refuse* | *The cat refused **to eat** its food.* |
| *pretend* | *The child pretended **to be** asleep.* |
| *wish* | *I wish **to speak** with the manager.* |

---

### C. Verb + Object + `to` (Infinitive with Object)

Some verbs can be followed by an object and then the infinitive form.

> **Structure:** *Verb + Object + to + V1*

| Verb | Example |
|------|---------|
| *tell* | *She told me **to close** the door.* |
| *ask* | *He asked me **to help** him.* |
| *want* | *I want you **to come** with me.* |
| *advise* | *The doctor advised her **to rest**.* |
| *allow* | *They won't allow us **to enter** without a ticket.* |
| *remind* | *Please remind me **to call** him tomorrow.* |
| *teach* | *My father taught me **to swim**.* |
| *encourage* | *The teacher encouraged the students **to participate**.* |
| *order* | *The captain ordered his soldiers **to advance**.* |
| *permit* | *The guard permitted them **to pass**.* |
| *require* | *The policy requires employees **to wear** a uniform.* |
| *expect* | *We expect you **to finish** the work by Friday.* |

---

### D. Preposition + `-ing`

Prepositions are almost always followed by a **gerund** (`-ing` form) when a verb is involved.

> **Structure:** *Preposition + -ing*

| Phrase | Example |
|--------|---------|
| *interested in* | *He is interested in **learning** about history.* |
| *good at* | *She is very good at **solving** problems.* |
| *bad at* | *I'm bad at **remembering** names.* |
| *think about/of* | *I'm thinking about **changing** my job.* |
| *dream of* | *He dreams of **becoming** a doctor.* |
| *instead of* | *I had tea instead of **coffee**.* (noun) / *Instead of **going** out, we stayed home.* |
| *insist on* | *We insisted on **paying** for the meal.* |
| *what about* | *What about **going** for a walk?* |
| *thank you for* | *Thank you for **helping** me.* |
| *before / after* | *Please check your work before **submitting** it.* / *She felt better after **talking** to her friend.* |

---

### E. Verb + Preposition + `-ing`

This is a specific case of the previous rule, where a verb is followed by a preposition, and then a gerund.

| Verb + Preposition | Example |
|--------------------|---------|
| *apologize for* | *He apologized for **being** late.* |
| *accuse of* | *She accused him of **lying**.* |
| *blame for* | *They blamed the company for **causing** the accident.* |
| *complain about* | *He complained about **the noise**.* (noun) / *...about **being** ignored.* |
| *insist on* | *I insist on **speaking** to the manager.* |
| *object to* | *I object to **paying** such a high price.* |
| *succeed in* | *She succeeded in **passing** the exam.* |
| *look forward to* | *I look forward to **hearing** from you.* ("to" is a preposition here) |
| *get used to / be used to* | *I am used to **waking** up early.* ("to" is a preposition) |

---

### F. Adjective + `to` (Infinitive)

Certain adjectives are followed by the infinitive form.

> **Structure:** *Adjective + to + V1*

| Adjective | Example |
|-----------|---------|
| *happy to* | *I'm happy **to help**.* |
| *glad to* | *We are glad **to see** you.* |
| *sorry to* | *I'm sorry **to hear** that.* |
| *afraid to* | *He was afraid **to ask** for a raise.* |
| *eager to* | *They are eager **to start** the project.* |
| *ready to* | *Are you ready **to go**?* |
| *difficult to* | *This book is difficult **to understand**.* |
| *easy to* | *The test was easy **to complete**.* |
| *important to* | *It is important **to listen** carefully.* |
| *possible to* | *It may be possible **to fix** this.* |
| *likely to* | *She is likely **to succeed**.* |

---

### G. Specific Structures and Exceptions

#### 1. Prefer, Would Rather, Would Prefer

| Structure | Example |
|-----------|---------|
| *prefer + gerund* (comparing activities) | *I prefer **reading** to watching TV.* |
| *prefer + infinitive* | *I prefer **to read** rather than watch TV.* |
| *would rather + base verb* | *I'd rather **stay** home tonight.* |
| *would prefer + to-infinitive* | *I'd prefer **to leave** now.* |
| *would prefer + object + to-infinitive* | *I'd prefer you **to help** me.* |

#### 2. Be / Get Used To

These structures use *to* as a **preposition**, so they are followed by a **gerund**.

| Structure | Meaning | Example |
|-----------|---------|---------|
| *be used to + -ing* | accustomed to something | *She is used to **working** long hours.* |
| *get used to + -ing* | become accustomed to something | *He got used to **waking** up early.* |

#### 3. There's No Point In / It's No Use

Express that something is useless or futile.

- *There's no point in **worrying** about things you can't control.*
- *It's no use **complaining**; nothing will change.*

#### 4. Purpose Clauses

| Structure | Example |
|-----------|---------|
| *to + V1* (purpose) | *I saved money **to buy** a car.* |
| *for + -ing* (purpose of an object) | *This is a good tool **for cutting** wood.* |
| *so that + clause* (with modal verb) | *Speak clearly **so that** everyone can understand you.* |

#### 5. Afraid To Do vs. Afraid Of Doing

| Structure | Meaning | Example |
|-----------|---------|---------|
| *afraid to do* (infinitive) | Fear of performing the action itself | *He is afraid **to climb** the ladder.* |
| *afraid of doing* (gerund) | Fear of the result or consequence | *He is afraid **of falling** from the ladder.* |

#### 6. Verbs of Perception: See, Hear, Watch, Feel

These verbs can be followed by either the **base verb** (bare infinitive) or the **-ing form**.

| Structure | Meaning | Example |
|-----------|---------|---------|
| *see/hear/watch/feel + object + V1* | Perception of the **whole action** | *I saw him **leave** the building.* (complete action) |
| *see/hear/watch/feel + object + -ing* | Perception of an action **in progress** | *I saw him **leaving** the building.* (in the process) |

#### 7. `-ing` Clauses (Participial Phrases)

Participial phrases act like adjectives, modifying nouns or pronouns.

| Type | Structure | Example |
|------|-----------|---------|
| Present Participle (active) | *-ing* phrase | *The man **standing over there** is my uncle.* / ***Feeling tired**, she went to bed early.* |
| Past Participle (passive) | *-ed* phrase | *The car **parked outside** is mine.* / ***Disappointed** with the results, he studied harder.* |

---

### H. Verbs with Two Objects (Ditransitive Verbs)

Some verbs can take **two objects**: a **direct object** (the thing) and an **indirect object** (the person). These objects can be ordered in two ways.

> **Pattern 1:** *Verb + Indirect Object (person) + Direct Object (thing)*
> **Pattern 2:** *Verb + Direct Object (thing) + to/for + Indirect Object (person)*

#### Common Ditransitive Verbs

| Verb | Pattern 1 (IO + DO) | Pattern 2 (DO + to/for + IO) |
|------|---------------------|------------------------------|
| *give* | *She gave **me** a book.* | *She gave a book **to me**.* |
| *send* | *They sent **us** a letter.* | *They sent a letter **to us**.* |
| *tell* | *He told **me** a story.* | *He told a story **to me**.* |
| *show* | *She showed **him** the way.* | *She showed the way **to him**.* |
| *offer* | *They offered **her** a job.* | *They offered a job **to her**.* |
| *lend* | *Can you lend **me** $10?* | *Can you lend $10 **to me**?* |
| *write* | *She wrote **me** an email.* | *She wrote an email **to me**.* |
| *buy* | *He bought **his wife** a gift.* | *He bought a gift **for his wife**.* |
| *make* | *She made **us** dinner.* | *She made dinner **for us**.* |
| *get* | *Can you get **me** a coffee?* | *Can you get a coffee **for me**?* |
| *cook* | *He cooked **his family** a meal.* | *He cooked a meal **for his family**.* |
| *find* | *I found **her** a job.* | *I found a job **for her**.* |

#### Key Rules for Two Objects

| Rule | Example |
|------|---------|
| **Pronouns are often placed first** (Pattern 1) | *Give **me** the book.* (not *Give the book to me* - though both are correct) |
| **With *to*:** verbs of transfer/giving | *give, send, lend, offer, show, tell, write, pass* |
| **With *for*:** verbs of doing for someone | *buy, make, get, cook, find, choose, pick* |
| **Cannot separate the verb from the direct object** (Pattern 2) | *She gave **a book** to me.* (DO before *to*) |

#### Passive Voice with Two Objects

Sentences with two objects can have **two passive forms**.

| Active | Passive (IO as subject) | Passive (DO as subject) |
|--------|------------------------|------------------------|
| *She gave me a book.* | *I **was given** a book.* | *A book **was given** to me.* |
| *They offered her a job.* | *She **was offered** a job.* | *A job **was offered** to her.* |
| *He bought her a gift.* | *She **was bought** a gift.* | *A gift **was bought** for her.* |

---

### I. Verbs Followed by Either Gerund or Infinitive (With Meaning Change)

Some verbs can be followed by both a gerund and an infinitive, but the **meaning changes**.

| Verb | + Gerund (-ing) | + Infinitive (to + V1) |
|------|----------------|------------------------|
| *stop* | *She stopped **smoking**.* (quit the action) | *She stopped **to smoke**.* (stopped in order to smoke) |
| *remember* | *I remember **locking** the door.* (memory of past action) | *Remember **to lock** the door.* (don't forget future action) |
| *forget* | *I'll never forget **meeting** her.* (memory of past event) | *Don't forget **to call** him.* (don't forget future action) |
| *regret* | *I regret **telling** him.* (sorry about past action) | *I regret **to inform** you.* (sorry about what I'm about to say) |
| *try* | *Try **pressing** the button.* (experiment with a method) | *Try **to fix** it.* (make an effort to achieve something) |
| *mean* | *Success means **working** hard.* (involves/requires) | *I didn't mean **to hurt** you.* (intend) |
| *need* | *The car needs **washing**.* (needs to be done - passive meaning) | *I need **to wash** the car.* (active obligation) |
| *go on* | *He went on **talking** for hours.* (continued) | *She went on **to become** a doctor.* (proceeded to next action) |

---

### J. Verbs Followed by Either Gerund or Infinitive (No/Negligible Meaning Change)

| Verb | Both are possible with little or no change in meaning |
|------|------------------------------------------------------|
| *begin* | *It began **raining**.* / *It began **to rain**.* |
| *start* | *She started **crying**.* / *She started **to cry**.* |
| *continue* | *He continued **working**.* / *He continued **to work**.* |
| *like* | *I like **swimming**.* / *I like **to swim**.* (slight nuance: gerund = enjoyment, infinitive = habit/choice) |
| *love* | *She loves **dancing**.* / *She loves **to dance**.* |
| *hate* | *I hate **being** late.* / *I hate **to be** late.* |
| *prefer* | *I prefer **reading**.* / *I prefer **to read**.* |

> **Note for *like/love/hate/prefer*:** In American English, the infinitive is more common for specific situations; the gerund is more common for general statements. However, both are widely used and accepted.

### L. Common Errors to Avoid

| Error | Correction |
|-------|------------|
| *I look forward **to see** you.* | *I look forward **to seeing** you.* |
| *She suggested **to go** home.* | *She suggested **going** home.* |
| *I enjoy **to read**.* | *I enjoy **reading**.* |
| *He made me **to laugh**.* | *He made me **laugh**.* (no *to* after *make*) |
| *I let him **to go**.* | *I let him **go**.* (no *to* after *let*) |
| *She gave **to me** a book.* (wrong order with *to*) | *She gave **me** a book.* or *She gave a book **to me**.* |
| *I stopped **to smoke** cigarettes.* (means you stopped in order to smoke) | *I stopped **smoking** cigarettes.* (to quit) |
| *I remembered **to lock** the door yesterday.* (if you actually have a memory) | *I remembered **locking** the door yesterday.* (recalling the action) |
