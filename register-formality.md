## 24. Register and Formality

Register refers to the level of formality or informality in language. Choosing the appropriate register is essential for effective communication. Using the wrong register can make you sound rude, awkward, or out of place.

> **Related Sections:** See *[Section 3. Short Forms (Contractions)](./short-forms-contractions.md)* for informal contractions, *[Appendix A: American vs. British English](./appendix-a-american-british-english.md)* for regional variations, and *[Section 20.1 Discourse Markers](./discourse-markers.md)* for conversational markers. (Also check [Appendix E: Abbreviations](./appendix-e-abbreviations.md)). 

---

### A. What Is Register?

Register is the variety of language you use depending on the situation, audience, and purpose. Think of it like choosing the right outfit – you wouldn't wear a tuxedo to the beach or shorts to a wedding.

| Situation | Appropriate Register |
|-----------|---------------------|
| Texting a friend | Informal |
| Writing a job application | Formal |
| Giving a presentation at work | Neutral / Semi-formal |
| Talking to a child | Informal, simple |
| Writing an academic paper | Very formal |

---

### B. The Three Main Registers

| Register | Also Called | Use | Examples |
|----------|-------------|-----|----------|
| **Formal** | Academic, Professional | Academic papers, business letters, official documents, speeches | *We are writing to inform you...* |
| **Neutral** | Standard, Semi-formal | Most everyday writing, news articles, work emails, instructions | *We want to let you know...* |
| **Informal** | Casual, Colloquial | Conversations with friends, text messages, personal emails, social media | *Just so you know...* |

---

### C. Formal Register

#### Characteristics

| Feature | Example |
|----------|---------|
| Complete sentences | *I am writing to apply for the position.* |
| No contractions | *I do not agree* (not *don't*) |
| Precise vocabulary | *I would like to inquire about...* (not *I want to ask about...*) |
| Impersonal structures | *It has been observed that...* (not *I noticed that...*) |
| Passive voice | *The experiment was conducted...* (not *We did the experiment...*) |
| Complex sentence structures | *Although the results were inconclusive, further research is warranted.* |
| No slang or idioms | Use standard expressions |
| Avoids phrasal verbs | *investigate* (not *look into*), *tolerate* (not *put up with*) |
| Full forms of words | *photograph* (not *photo*), *advertisement* (not *ad*) |

#### When to Use Formal Register

| Situation | Example |
|-----------|---------|
| Academic essays | *This paper argues that...* |
| Business letters | *Please find attached...* |
| Job applications | *I am writing to apply for the position of...* |
| Official documents | *The undersigned hereby agrees...* |
| Legal writing | *Pursuant to section 3.2...* |
| Academic presentations | *It is evident from the data that...* |
| Reports and proposals | *The findings indicate a significant correlation...* |

#### Formal vs. Informal Word Choices

| Informal | Formal |
|----------|--------|
| *ask* | *inquire, request* |
| *help* | *assist* |
| *need* | *require* |
| *start* | *commence* |
| *end* | *terminate* |
| *buy* | *purchase* |
| *tell* | *inform* |
| *get* | *receive, obtain* |
| *use* | *utilize* |
| *show* | *demonstrate* |
| *try* | *attempt* |
| *enough* | *sufficient* |
| *so* | *therefore, consequently* |
| *but* | *however, nevertheless* |
| *about* | *approximately, regarding* |
| *say* | *state, indicate* |
| *think* | *believe, consider* |
| *give* | *provide, furnish* |

#### Formal Sentence Examples

| Informal | Formal |
|----------|--------|
| *I'm sorry I'm late.* | *Please accept my apologies for my delay.* |
| *Can you help me?* | *I would appreciate your assistance.* |
| *Thanks for your email.* | *Thank you for your correspondence.* |
| *I need more time.* | *Additional time is required.* |
| *I want to apply for the job.* | *I wish to submit my application for the position.* |
| *Here's my opinion.* | *From my perspective, it would appear that...* |
| *Let me know if you need anything.* | *Please do not hesitate to contact me should you require further assistance.* |

### D. Informal Register

#### Characteristics

| Feature | Example |
|----------|---------|
| Contractions | *I don't know* (not *I do not know*) |
| Short, simple sentences | *Got your message. Call me later.* |
| Slang and idioms | *That's awesome!* / *I'm totally wiped out.* |
| Phrasal verbs | *figure out, put up with, go on* (not *determine, tolerate, continue*) |
| Personal pronouns | *I think, in my opinion, you know* |
| Ellipsis (omitting words) | *Coming?* (for *Are you coming?*) |
| Discourse markers | *well, so, anyway, you know, like* |
| Active voice (almost always) | *We conducted the experiment.* (not *was conducted*) |

#### When to Use Informal Register

| Situation | Example |
|-----------|---------|
| Talking to friends | *Hey! What's up?* |
| Text messages | *C u at 8. Don't be late.* |
| Personal emails | *Just wanted to let you know...* |
| Social media posts | *Loving this weather! ☀️* |
| Casual conversations | *So, what do you wanna do?* |
| Private messages and chats | *I'll be there in 5. OK?* |
| Personal blogs/vlogs | *So here's what happened to me yesterday...* |

#### Informal Sentence Examples

| Neutral | Informal |
|---------|----------|
| *I am going to leave now.* | *I'm gonna head out.* |
| *That is very good.* | *That's awesome / great / cool.* |
| *I do not understand the problem.* | *I don't get it.* |
| *Can you help me with this?* | *Can you give me a hand?* |
| *He is very tired.* | *He's totally wiped out.* |
| *I am very hungry.* | *I'm starving.* |

#### Common Informal Features

| Feature | Example |
|----------|---------|
| **Contractions** | *don't, can't, won't, I'm, you'll, they've* |
| **Gonna / wanna / gotta** | *I'm gonna go. / I wanna leave. / I gotta run.* |
| **Kinda / sorta** | *It's kinda weird.* |
| **Yeah / yep / nope** | *Yeah, I think so. / Nope, not me.* |
| **Exclamation marks** | *That's amazing!* |
| **Interjections** | *Wow! / Oops! / Hey! / Oh!* |
| **Abbreviations (texting)** | *LOL, OMG, BTW, FYI, IDK* |

### E. Neutral Register

#### Characteristics

| Feature | Example |
|----------|---------|
| Standard vocabulary | *I need to ask you something.* |
| Natural sentence structure | *Could you help me with this?* |
| Some contractions (optional) | Acceptable but not excessive |
| No slang or overly formal words | *I'd like to request...* (neutral) vs. *I want to ask...* (informal) vs. *I hereby request...* (formal) |
| Appropriate for most situations | Work emails, news articles, instructions |
| Balanced use of active/passive | Appropriate for context |

#### When to Use Neutral Register

| Situation | Example |
|-----------|---------|
| Work emails to colleagues | *Could you send me the report by Friday?* |
| News articles | *The president announced new policies yesterday.* |
| Instructions | *Press the button to start the machine.* |
| Most written English (non-academic) | *The study found a connection between diet and health.* |
| Classroom discussions (non-formal) | *I think the author is trying to say that...* |
| Everyday business communication | *Our meeting has been moved to 3 PM.* |

---

### F. Comparing the Three Registers

| Feature | Formal | Neutral | Informal |
|---------|--------|---------|----------|
| **Contractions** | No | Optional (limited) | Yes (frequent) |
| **Personal pronouns** | Rare (*one, the researcher*) | Common (*I, we, you*) | Very common |
| **Passive voice** | Common | Occasional | Rare (almost never) |
| **Phrasal verbs** | Rare (avoided) | Some | Common (preferred) |
| **Slang** | Never | Rare | Common |
| **Sentence length** | Long, complex | Medium | Short, simple |
| **Vocabulary** | Academic/Latin-derived | Standard | Colloquial/Anglo-Saxon |
| **Exclamation marks** | Never | Rare | Common |
| **Ellipsis** | No | Rare | Common |
| **Abbreviations** | Standard only (*e.g., i.e., etc.*) | Some | Many (including texting) |

#### Same Idea, Three Registers

| Formal | Neutral | Informal |
|--------|---------|----------|
| *I am unable to attend the meeting.* | *I can't make the meeting.* | *Sorry, gotta skip the meeting.* |
| *Please accept my sincere apologies.* | *I'm sorry.* | *My bad.* |
| *The committee has reached a decision.* | *The committee made a decision.* | *They've decided.* |
| *I would like to request your assistance.* | *Could you help me?* | *Can you give me a hand?* |
| *The project was completed ahead of schedule.* | *We finished the project early.* | *We got it done early.* |
| *Pursuant to our conversation...* | *Following up on our conversation...* | *Just checking in about what we talked about...* |
| *I am writing to inform you that...* | *Just letting you know that...* | *Hey, just so you know...* |

### G. Register in Speech vs. Writing

| Aspect | Written Language | Spoken Language |
|--------|------------------|-----------------|
| **Typical register** | More formal, neutral | More informal, often casual |
| **Sentence structure** | Complete, well-formed | Fragments, repairs, false starts |
| **Vocabulary** | Precise, varied | Everyday, repetitive, vague |
| **Features** | Passive voice, nominalisation | Pauses, fillers (*um, well, like*) |
| **Contractions** | Often avoided (formal) | Very common |
| **Example** | *The results indicate a significant increase.* | *So the results show, like, a big increase, you know?* |

**Spoken variants of formal registers:**
- Academic lectures -> *As we can see from the data...*
- Business presentations -> *What I'd like to focus on today is...*
- Courtroom language (still distinct) -> *Your Honor, I would like to submit...*

### H. Common Register Mistakes

#### 1. Being Too Formal in Casual Situations

| Too Formal (Awkward) | Better (Natural) |
|----------------------|------------------|
| *To whom it may concern, I am writing to inform you that I will be late.* (to a friend) | *Hey, gonna be late. Sorry!* |
| *I would appreciate it if you could pass the salt.* (at dinner) | *Could you pass the salt?* |
| *I am experiencing a state of hunger.* | *I'm hungry.* |
| *I would like to inquire as to the time.* | *What time is it?* |

#### 2. Being Too Informal in Formal Situations

| Too Informal (Unprofessional) | Better (Appropriate) |
|------------------------------|----------------------|
| *Hey, I wanna apply for the job.* (job application) | *Dear Sir/Madam, I wish to apply for the position.* |
| *Thanks for the email. Here's my stuff.* (business) | *Thank you for your correspondence. Please find attached the requested documents.* |
| *Sorry I'm late. My bad.* (formal meeting) | *Please accept my apologies for my delay.* |
| *OK, so basically, here's what we found out.* (academic presentation) | *The findings of this study indicate the following.* |

#### 3. Mixing Registers Inappropriately

| Mixed (Inconsistent) | Consistent |
|---------------------|------------|
| *I am writing to inquire about the job 'cause I need work.* | *I am writing to inquire about the position as I am currently seeking employment.* |
| *The experiment was conducted, and we got some cool results.* | *The experiment was conducted, and the results were significant.* |
| *It is imperative that you, like, finish the report.* | *It is imperative that you finish the report.* |

### I. Email Formatting (Register-Specific)

Email register varies significantly based on the recipient and purpose.

#### 1. Email Subject Lines

| Register | Example |
|----------|---------|
| **Formal** | *Application for Marketing Manager Position – John Smith* |
| **Neutral** | *Meeting rescheduled to Friday, March 15* |
| **Informal** | *Quick question / Catching up / Fwd: article you might like* |

| Rule | Formal | Neutral | Informal |
|------|--------|---------|----------|
| Capitalize all major words | ✓ | Optional | Optional |
| Include specific details (dates, names) | ✓ | ✓ | No |
| Use complete phrases | ✓ | Sometimes | Rarely |

#### 2. Email Greetings (Salutations)

| Recipient | Register | Example |
|-----------|----------|---------|
| Unknown recipient | Formal | *Dear Sir or Madam,* / *To whom it may concern,* |
| Known person (formal) | Formal | *Dear Dr. Johnson,* / *Dear Mr. Smith,* |
| Colleague (neutral) | Neutral | *Hello Sarah,* / *Hi John,* |
| Close colleague | Neutral | *Hi Maria,* / *Good morning team,* |
| Friend / family | Informal | *Hey Tom,* / *Hi Mom,* / *Sarah,* |

**Punctuation after greetings:**
- American English: Use a colon (`:`) after formal greetings; comma (`,`) after informal
- British English: Often no punctuation after the greeting

| Register | American | British |
|----------|----------|---------|
| Formal | *Dear Mr. Smith:* | *Dear Mr. Smith* |
| Neutral | *Hello Sarah,* | *Hello Sarah* |
| Informal | *Hey Tom,* | *Hey Tom* |

#### 3. Email Body (Content and Tone)

| Register | Language Features | Example |
|----------|-------------------|---------|
| **Formal** | Passive voice, full forms, no contractions, polite requests | *I am writing to inquire about the status of my application. Would you please provide an update at your earliest convenience?* |
| **Neutral** | Active voice, some contractions, direct but polite | *I'm writing to check on my application. Could you let me know if there's an update?* |
| **Informal** | Contractions, phrasal verbs, casual tone, shorter sentences | *Just checking in on my application. Any news yet? Thanks!* |

#### 4. Email Closings

| Register | Examples |
|----------|----------|
| **Formal** | *Yours faithfully,* (when name unknown) / *Yours sincerely,* (when name known) / *Respectfully,* |
| **Neutral** | *Best regards,* / *Kind regards,* / *Sincerely,* / *Thank you,* |
| **Informal** | *Best,* / *Cheers,* / *Thanks,* / *Talk soon,* / *Take care,* / *Warmly,* |

**Closing rules:**
- *Yours faithfully* – Use when salutation is "Dear Sir/Madam" (name unknown)
- *Yours sincerely* – Use when salutation is "Dear Mr. Smith" (name known)
- *Best regards* – Neutral, widely accepted
- *Cheers* – Informal, common in British/Australian English, less common in US

#### 5. Complete Email Examples

**Formal Email:**

```
Subject: Application for Research Assistant Position

Dear Dr. Williams,

I am writing to apply for the Research Assistant position advertised on your university's website.

Please find attached my curriculum vitae and cover letter for your consideration. I would be grateful if you could confirm receipt of this application.

Thank you for your time and consideration.

Yours sincerely,

John Smith
```

**Neutral Email:**

```
Subject: Meeting rescheduled to Friday

Dear Dr. Williams,

I'm writing to let you know that our meeting has been rescheduled to Friday at 2 PM.

Please find the updated calendar invitation attached. Let me know if this time still works for you.

Best regards,

John Smith
```

**Informal Email:**

```
Hey Jen,

Just wanted to check in about Friday's meeting – it's been moved to 2 PM. Let me know if that still works for you!

Thanks,

John
```

### J. Letter Writing Format (Business and Personal Letters)

Traditional letter writing follows specific format conventions that vary by register.

#### 1. Business Letter (Formal) Format

**Standard Business Letter Layout:**

```
[Your Address]
123 Main Street
New York, NY 10001

[Date]
March 15, 2024

[Recipient's Name and Title]
Ms. Jane Johnson
Human Resources Manager
XYZ Corporation
456 Corporate Drive
Chicago, IL 60606

[Salutation]
Dear Ms. Johnson:

[Subject Line (optional)]
RE: Application for Marketing Position

[Body of Letter]
I am writing to express my interest in the Marketing Manager position advertised on LinkedIn...

[Closing]
Sincerely,

[Signature (handwritten if physical letter)]
[Typed Name]
John Smith
```

**Key Elements of Business Letters:**

| Element | Format | Example |
|---------|--------|---------|
| **Your address** | Top right or left (block format uses left) | *123 Main Street, New York, NY 10001* |
| **Date** | Below your address | *March 15, 2024* |
| **Inside address** | Recipient's full name, title, company, address | *Ms. Jane Johnson, HR Manager...* |
| **Salutation** | *Dear* + title + last name + colon (US) | *Dear Ms. Johnson:* |
| **Subject line** | Optional, often underlined or bold | *RE: Job Application* |
| **Body** | Single-spaced, double-spaced between paragraphs | *I am writing to...* |
| **Closing** | Capitalize first word only + comma | *Sincerely,* / *Yours truly,* |
| **Signature** | Handwritten above typed name | [signature] / *John Smith* |

**Common Business Letter Closings:**
- *Yours truly,* – Standard
- *Sincerely,* – Standard
- *Respectfully,* – Formal (authority figures)
- *Yours faithfully,* – British (when name unknown)
- *Yours sincerely,* – British (when name known)

#### 2. Personal Letter (Informal) Format

**Standard Personal Letter Layout:**

```
[Your Address (optional, or just date)]
123 Main Street
New York, NY 10001

[Date]
March 15, 2024

[Salutation]
Dear Sarah,

[Body of Letter]
I hope you're doing well. It's been so long since we last caught up! I'm writing to tell you about my recent trip to Italy...

[Closing]
Love,

[Signature]
John
```

**Key Elements of Personal Letters:**

| Element | Format | Example |
|---------|--------|---------|
| **Your address** | Often omitted in personal letters | *123 Main Street* (optional) |
| **Date** | Usually included | *March 15, 2024* |
| **Salutation** | *Dear* + first name + comma | *Dear Sarah,* |
| **Body** | Conversational, contractions, personal tone | *I hope you're doing well...* |
| **Closing** | Warm, personal | *Love,* / *Best,* / *Your friend,* |
| **Signature** | First name only | *John* |

**Common Personal Letter Closings:**
- *Love,* – Family, romantic partner, close friends
- *With love,* – Same as above
- *Best,* – Friends, acquaintances
- *Warmly,* – Friends, family
- *Take care,* – General
- *Talk soon,* – Friends, colleagues
- *Yours,* – Friendly, slightly formal
- *Affectionately,* – Family, close friends

#### 3. Letter Formats by Register

| Feature | Formal (Business) | Neutral (Semi-formal) | Informal (Personal) |
|---------|-------------------|----------------------|--------------------|
| **Your address** | Always included | Often included | Optional |
| **Inside address** | Always included | Usually included | Omitted |
| **Subject line** | Often included | Optional | Rarely |
| **Salutation** | *Dear Mr./Ms./Dr.* | *Dear First Name* | *Dear First Name* or *Hi* |
| **Closing** | *Sincerely, / Yours truly,* | *Best regards,* | *Love, / Best, / Cheers* |
| **Contractions** | Avoided | Some | Many |
| **Personal pronouns** | Rare | Some | Many |
| **Typed/signed name** | Full name | First name | First name |

### L. Adjusting Register in Speech and Writing

#### Ask Yourself These Questions

| Question | Helps Determine |
|----------|-----------------|
| *Who is my audience?* | Friend? Colleague? Boss? Stranger? |
| *What is my purpose?* | Inform? Persuade? Request? Apologize? |
| *What is the medium?* | Text? Email? Speech? Letter? |
| *What is the setting?* | Casual? Professional? Academic? |
| *What is my relationship to the audience?* | Close? Distant? Unknown? |

#### Quick Guide: Formal vs. Informal Features

| Feature | Formal | Informal |
|----------|--------|----------|
| Verb form | *do not* | *don't* |
| Vocabulary | *request, inquire* | *ask, want* |
| Pronouns | *one, the researcher* | *I, you, we* |
| Sentence start | *It is important to note that...* | *So, anyway...* |
| Endings | *Sincerely, Yours faithfully* | *Cheers, Best, Talk soon* |
| Passive voice | *The results were analyzed* | *We looked at the results* |
| Address form | *Dear Mr. Smith* | *Hey John* |

### M. Register Conversion (Practice)

Learn to shift the same message between registers.

| Register | Version |
|----------|---------|
| **Formal** | *We regret to inform you that your application has not been successful on this occasion.* |
| **Neutral** | *We're sorry to tell you that you didn't get the job this time.* |
| **Informal** | *Sorry, you didn't get the gig.* |

| Register | Version |
|----------|---------|
| **Formal** | *I would be grateful if you could provide me with the relevant information at your earliest convenience.* |
| **Neutral** | *Could you please send me the information when you get a chance?* |
| **Informal** | *Can you let me know? Thanks!* |

| Register | Version |
|----------|---------|
| **Formal** | *The meeting has been postponed until further notice.* |
| **Neutral** | *The meeting is off for now. We'll let you know when it's rescheduled.* |
| **Informal** | *The meeting's cancelled. More info later.* |

| Register | Version |
|----------|---------|
| **Formal** | *Thank you for your prompt attention to this matter.* |
| **Neutral** | *Thanks for your help with this.* |
| **Informal** | *Thanks a lot!* |

### N. Quick Reference Summary

| Register | When to Use | Key Features | Example |
|----------|-------------|--------------|---------|
| **Formal** | Academic, business, legal, official | No contractions, passive voice, complex sentences, precise vocabulary, no slang | *We are writing to inform you that your application has been approved.* |
| **Neutral** | Everyday writing, news, work emails | Standard vocabulary, natural structure, some contractions, balanced active/passive | *Your application has been approved.* |
| **Informal** | Friends, family, casual conversation | Contractions, slang, short sentences, phrasal verbs, discourse markers, ellipsis | *Hey, good news – you got it!* |

### O. Common Errors to Avoid

| Error | Correction |
|-------|------------|
| Using *gonna/wanna* in formal writing | Use *going to / want to* |
| Using *thanks* in a formal letter | Use *Thank you for your consideration* |
| Using *Dear Sir* when you know the name | Use *Dear Ms. Jones* |
| Starting a formal email with *Hey* | Use *Dear Mr./Ms. [Last Name]* |
| Using no greeting in a professional email | Always include a greeting |
| Overusing exclamation marks in neutral writing | Use sparingly or not at all |
| Using slang in a job application | Avoid completely |
| Using texting abbreviations (*LOL, OMG, BTW*) in business emails | Avoid; write full words |
| Using *you guys* in formal speech | Use *ladies and gentlemen* or *everyone* |
| Starting a formal letter with *Hello* | Use *Dear* + title + last name |
| Using *Yours faithfully* with a named recipient | Use *Yours sincerely* when you know the name |
| Omitting the subject line in a formal email | Always include a clear subject line |
| Using *Best* or *Cheers* in a formal business letter | Use *Sincerely* or *Yours truly* |
