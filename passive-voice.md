## 8. Passive Voice

The passive voice is used when the focus is on the action itself, not on who or what performs it. The object of an active sentence becomes the subject of a passive sentence, and the doer (agent) is often omitted when unknown, unimportant, or obvious.

- **Active:** Mehdi broke the window.
- **Passive:** The window was broken (by Mehdi).

> **Key Feature:** All passive tenses end in the past participle (V3).

---

### A. Formation Basics

The passive voice is formed using the appropriate tense of **be** + past participle (V3).

#### Conjugation of "to be"

| Tense | Forms |
|-------|-------|
| **Base Form** | be |
| **Present** | am (I) / is (he, she, it) / are (you, we, they) |
| **Past** | was (I, he, she, it) / were (you, we, they) |
| **Future** | will be (all subjects) |
| **Present Participle** | being |
| **Past Participle** | been |
| **Present Perfect** | has been (he, she, it) / have been (I, you, we, they) |
| **Past Perfect** | had been (all subjects) |
| **Future Perfect** | will have been (all subjects) |

#### Conjugation of "to have" (for perfect tenses)

| Tense | Forms |
|-------|-------|
| **Base Form** | have |
| **Present** | has (he, she, it) / have (I, you, we, they) |
| **Past** | had (all subjects) |
| **Future** | will have (all subjects) |
| **Present Participle** | having |
| **Past Participle** | had |
| **Present Perfect** | has had (he, she, it) / have had (I, you, we, they) |
| **Past Perfect** | had had (all subjects) |
| **Future Perfect** | will have had (all subjects) |

#### Conjugation of "to do" (for questions, negatives, emphasis)

| Tense | Forms |
|-------|-------|
| **Base Form** | do |
| **Present** | does (he, she, it) / do (I, you, we, they) |
| **Past** | did (all subjects) |
| **Future** | will do (all subjects) |
| **Present Participle** | doing |
| **Past Participle** | done |
| **Present Perfect** | has done (he, she, it) / have done (I, you, we, they) |
| **Past Perfect** | had done (all subjects) |
| **Future Perfect** | will have done (all subjects) |

---

### B. Simple Tenses (be + V3)

| Tense | Form | Examples |
|-------|------|----------|
| Infinitive Passive | (to) be + V3 | *The work needs **to be done**.* / *The problem seems **to be solved**.* |
| Present Simple | am/is/are + V3 | *English **is spoken** in many countries.* / *Mistakes **are made**.* / *The door **is opened** by the cleaner every morning.* |
| Past Simple | was/were + V3 | *The window **was broken** yesterday.* / *They **were invited** to the party.* / *The book **was written** last year.* |
| Future Simple | will + be + V3 | *The package **will be delivered** tomorrow.* / *The report **will be finished** by Friday.* |
| Future (going to) | am/is/are + going to be + V3 | *The car **is going to be sold**.* / *The problem **is going to be fixed**.* |

---

### C. Continuous Tenses (be + being + V3)

| Tense | Form | Examples |
|-------|------|----------|
| Present Continuous | am/is/are + being + V3 | *The house **is being built**.* / *The food **is being prepared**.* / *His actions **are being watched**.* |
| Past Continuous | was/were + being + V3 | *The bridge **was being repaired** when the storm hit.* / *They **were being interviewed** at that time.* |
| Future Continuous | will + be + being + V3 | *The report **will be being written** at this time tomorrow.* (rare) |

> **Note:** Future continuous passive is grammatically correct but rarely used in practice. Native speakers usually rephrase.

---

### D. Perfect Tenses (have/has/had + been + V3)

| Tense | Form | Examples |
|-------|------|----------|
| Present Perfect | have/has + been + V3 | *The email **has been sent**.* / *The project **has been completed**.* / *Many mistakes **have been made**.* |
| Past Perfect | had + been + V3 | *The problem **had been solved** before we arrived.* / *They realized the car **had been stolen**.* |
| Future Perfect | will + have + been + V3 | *The contract **will have been signed** by Friday.* / *The work **will have been finished** by then.* |

---

### E. Perfect Continuous Tenses (have/has/had + been + being + V3)

| Tense | Form | Example |
|-------|------|---------|
| Present Perfect Continuous | have/has + been + being + V3 | *The house **has been being renovated** for months.* |
| Past Perfect Continuous | had + been + being + V3 | *The car **had been being repaired** for a week.* |

> **Note:** These forms are grammatically correct but extremely rare and awkward. Native speakers avoid them by rephrasing (e.g., *They have been renovating the house for months.*)

---

### F. Modal Verbs in Passive (modal + be + V3)

*Form:* modal verb + be + past participle

| Modal | Example |
|-------|---------|
| *can* | *This car **can be rented** by the hour.* |
| *could* | *That **could be seen** from a distance.* |
| *may* | *The results **may be published** next week.* |
| *might* | *The work **might be finished** soon.* |
| *must* | *The rules **must be followed**.* |
| *should* | *You **should be careful**.* |
| *would* | *The problem **would be solved** if we had more time.* |
| *ought to* | *This **ought to be done** immediately.* |
| *used to* | *Cars **used to be made** differently.* |

#### Modal + Perfect Passive

*Form:* modal + have + been + V3

| Modal | Example |
|-------|---------|
| *should have* | *The work **should have been finished** yesterday.* |
| *could have* | *It **could have been avoided**.* |
| *must have* | *The key **must have been left** at home.* |
| *might have* | *The package **might have been delivered** already.* |
| *would have* | *The building **would have been destroyed** if the fire had spread.* |

---

### G. Personal vs. Impersonal Passive

#### Personal Passive

The **direct object** of an active sentence becomes the **subject** of the passive sentence.

| Active | Passive |
|--------|---------|
| *The chef **prepared the meal**.* | ***The meal** was prepared by the chef.* |
| *The committee **will announce the results**.* | ***The results** will be announced by the committee.* |
| *Someone **stole my wallet**.* | ***My wallet** was stolen.* |

#### Impersonal Passive

Used with verbs that do not take a direct object or when generalizing opinions/beliefs.

##### Using "it" as a Dummy Subject (with verbs of saying, believing, thinking, reporting)

| Active | Impersonal Passive |
|--------|-------------------|
| *People say that honesty is the best policy.* | *It is said that honesty is the best policy.* |
| *They believe that the project will succeed.* | *It is believed that the project will succeed.* |
| *Authorities report that the situation is under control.* | *It is reported that the situation is under control.* |
| *Everyone knows that he is lying.* | *It is known that he is lying.* |
| *They expect that the president will resign.* | *It is expected that the president will resign.* |

**Common verbs used in this pattern:**
- *say, believe, think, know, report, expect, understand, assume, claim, consider, estimate, feel, find, hope, presume, recommend, suggest, suppose*

##### Using the Indirect Object as the Subject (with verbs like *tell, ask, give, show, teach, offer, pay, promise, send*)

| Active | Impersonal Passive |
|--------|-------------------|
| *Someone told me that the meeting was cancelled.* | *I was told that the meeting was cancelled.* |
| *The teacher gave the students a difficult assignment.* | *The students were given a difficult assignment.* |
| *They offered her a promotion.* | *She was offered a promotion.* |
| *He promised us a refund.* | *We were promised a refund.* |
| *The company paid him a bonus.* | *He was paid a bonus.* |

---

### H. Two Passive Forms from One Active Sentence

When an active sentence has both a **direct object** (thing) and an **indirect object** (person), two different passive sentences are possible.

| Active | Passive (Indirect Object as Subject) | Passive (Direct Object as Subject) |
|--------|--------------------------------------|-----------------------------------|
| *The company offered **him** a **position**.* | *He was offered a position.* | *A position was offered to him.* |
| *She gave **me** a **book**.* | *I was given a book.* | *A book was given to me.* |
| *They sent **us** an **invitation**.* | *We were sent an invitation.* | *An invitation was sent to us.* |
| *The teacher showed **the students** a **film**.* | *The students were shown a film.* | *A film was shown to the students.* |

> **Note:** Not all verbs allow both forms. Verbs like *explain, describe, announce, suggest, mention* do not typically allow the indirect object to become the subject:
> - Active: *She explained the problem to me.*
> - ✓ *The problem was explained to me.*
> - ✗ *I was explained the problem.* (incorrect)

---

### I. Reporting Verbs in Passive (Advanced)

There are two main patterns for reporting verbs in the passive:

#### Pattern 1: *It + passive + that-clause*

| Active | Passive |
|--------|---------|
| *People say that he is a genius.* | *It is said that he is a genius.* |
| *They believe that she lived here.* | *It is believed that she lived here.* |
| *Everyone knows that he is honest.* | *It is known that he is honest.* |

#### Pattern 2: *Subject + passive + to-infinitive*

| Active | Passive |
|--------|---------|
| *People say that he is a genius.* | *He is said to be a genius.* |
| *They believe that she lived here.* | *She is believed to have lived here.* |
| *Everyone knows that he was a spy.* | *He is known to have been a spy.* |
| *They expect that the company will grow.* | *The company is expected to grow.* |
| *Authorities report that the thief has escaped.* | *The thief is reported to have escaped.* |

**Tense changes in the infinitive:**

| Time Reference | Infinitive Form | Example |
|----------------|-----------------|---------|
| Same time as reporting verb | *to be* + adjective/noun | *He is said **to be** wealthy.* |
| Earlier than reporting verb | *to have* + V3 | *She is believed **to have left**.* |
| Future relative to reporting verb | *to be going to* / *to be to* | *The company is expected **to be going to expand**.* |

#### *Be Supposed To* (Expectation or Obligation)

| Use | Example |
|-----|---------|
| **Expectation** | *The train **is supposed to arrive** at 9 AM.* (It is expected) |
| **Obligation** | *You **are supposed to wear** a uniform.* (You should) |
| **Reported belief** | *He **is supposed to be** very rich.* (People say) |

---

### J. The Agent (by Phrase)

The *by* phrase identifies the performer of the action. Include it when the agent is known and important; omit it when unknown, unimportant, or obvious.

| Situation | Example |
|-----------|---------|
| **Agent known and important** | *The Mona Lisa was painted **by Leonardo da Vinci**.* / *The email was sent **by Sarah**.* |
| **Agent unknown** | *My bike was stolen.* / *The window has been broken.* |
| **Agent unimportant/obvious** | *The road is being repaired.* / *The meeting will be held in the conference room.* |
| **Agent implied through context** | *The patient was operated on.* (implied: a surgeon) |
| **To avoid responsibility/blame** | *Mistakes were made.* (without saying who) |

---

### K. Have Something Done (Causative)

*Form:* have/had + object + past participle (V3)

Used when someone **arranges for something to be done** by another person (causative). Also used for **unfortunate events** that happen to someone.

| Use | Example |
|-----|---------|
| **Service/Repair (arranged by the subject)** | *I **had my hair cut** yesterday.* (Someone else cut my hair) |
| **Service/Repair (arranged)** | *She **is having her car repaired**.* (A mechanic is repairing it) |
| **Unfortunate event (not arranged)** | *He **had his phone stolen**.* (Someone stole his phone) |
| **Unfortunate event** | *They **had their house broken into**.* |

#### Alternative: *Get Something Done* (informal)

In informal English, *get* can replace *have* in the causative structure.

| Formal (have) | Informal (get) |
|---------------|----------------|
| *I need to **have** my car fixed.* | *I need to **get** my car fixed.* |
| *She **had** her hair cut.* | *She **got** her hair cut.* |
| *We **had** the roof repaired.* | *We **got** the roof repaired.* |

---

### L. Passive with Prepositional Verbs and Phrasal Verbs

In passive sentences, prepositions remain attached to the verb.

| Active | Passive |
|--------|---------|
| *They **looked after** the child.* | *The child **was looked after**.* |
| *Someone **has taken care of** the problem.* | *The problem **has been taken care of**.* |
| *They **laughed at** him.* | *He **was laughed at**.* |
| *The doctor **operated on** her.* | *She **was operated on**.* |
| *Nobody **listened to** the warning.* | *The warning **was not listened to**.* |

> **Note:** The preposition stays with the verb and cannot be omitted.

---

### M. Passive with *-ing* and *to*-Infinitive

Passive forms can be used as gerunds (-ing) or infinitives (*to*).

#### Passive Gerund

| Form | Example |
|------|---------|
| *being + V3* (present) | *She dislikes **being told** what to do.* |
| *having been + V3* (past) | *He regretted **having been accused** unfairly.* |

#### Passive Infinitive

| Form | Example |
|------|---------|
| *to be + V3* (present) | *She expects **to be given** a promotion.* |
| *to have been + V3* (past) | *He seems **to have been told** the news already.* |

---

### N. Passive in Conditional Statements

| Conditional | Active | Passive |
|-------------|--------|---------|
| **First** | *If the company **hires** more staff, productivity will increase.* | *If more staff **are hired**, productivity will increase.* |
| **Second** | *If I **had** more time, I would learn a new language.* | *If more time **were given** to me, I would learn a new language.* |
| **Third** | *If the team **had won** the match, they would have advanced.* | *If the match **had been won**, they would have advanced.* |

---

### O. Passive in Imperative Statements

> **Structure:** *Let + object + be + past participle* (formal instructions)

| Direct Command | Passive Instruction |
|----------------|---------------------|
| *Lock the door.* | ***Let the door be locked**.* |
| *Check the engine.* | ***Let the engine be checked**.* |
| *Don't touch the wires.* | ***Let the wires not be touched**.* / *Do not let the wires be touched.* |

**Alternative (strong commands with modals):**
- *The report **must be submitted** by Friday.*
- *All guests **must be registered** at the reception.*

---

### P. When Not to Use the Passive

Some verbs cannot be used in the passive because they do not take a direct object (intransitive verbs).

| Verb Type | Example (Active only) | Passive Not Possible |
|-----------|----------------------|---------------------|
| Intransitive verbs (no object) | *She **arrived** late.* | ✗ *Late was arrived.* |
| | *The baby **cried**.* | ✗ *Was cried.* |
| **Verbs of being/seeming** | *She **is** a doctor.* | ✗ *A doctor is been by her.* |
| | *It **seems** wrong.* | ✗ *Wrong is seemed.* |
| **Reciprocal verbs** (with *each other*) | *They **agreed**.* | ✗ *Was agreed.* |
| **Verbs followed by certain prepositions** | *The house **belongs to** me.* | ✗ *I am belonged to.* |

---

### Q. Common Verbs That Are Rarely Used in the Passive

Some verbs can technically be passive but are rarely used that way in practice.

| Verb | Active Example | Passive (rare/awkward) |
|------|----------------|------------------------|
| *have* (possession) | *I have a car.* | ✗ *A car is had by me.* |
| *resemble* | *She resembles her mother.* | ✗ *Her mother is resembled by her.* |
| *lack* | *He lacks confidence.* | ✗ *Confidence is lacked by him.* |
| *suit* | *The dress suits you.* | ✗ *You are suited by the dress.* |
| *fit* | *The shoes fit me.* | ✗ *I am fitted by the shoes.* |

---

### R. Active vs. Passive: Choosing the Right Voice

#### Prefer Active Voice when:

| Reason | Example |
|--------|---------|
| **Directness and clarity** | *The team **completed** the project ahead of schedule.* |
| **Stronger impact** | *The company **launched** a new product.* |
| **Assigning responsibility** | *John **made** this mistake.* |
| **Avoiding wordiness** | Active sentences are usually shorter |

#### Prefer Passive Voice when:

| Reason | Example |
|--------|---------|
| **Receiver of action is more important** | *The archaeological site **was discovered** in 1922.* |
| **Agent is unknown** | *My wallet **has been stolen**.* |
| **Agent is unimportant or obvious** | *The results **will be announced** tomorrow.* |
| **Objectivity or formality (scientific/academic writing)** | *The samples **were analyzed** using a mass spectrometer.* |
| **Varying sentence structure** | Avoids repetitive subject-verb patterns |
| **Avoiding blame or responsibility** | *Mistakes **were made**.* |

### T. Common Errors to Avoid

| Error | Correction | Rule |
|-------|------------|------|
| *The window was break.* | *The window was **broken**.* | Passive always uses past participle (V3) |
| *She was offered a job?* (no auxiliary) | ***Was** she offered a job?* | Use auxiliary for passive questions |
| *The book written by her.* (missing auxiliary) | *The book **was** written by her.* | Passive needs form of *be* |
| *The letter is being send.* | *The letter is being **sent**.* | Use V3 (past participle), not V1 |
| *He is said that he is rich.* | *He is said **to be** rich.* | Use *to*-infinitive pattern |
| *It is said that he is rich.* | Correct – impersonal passive | ✓ |
| *I am agree.* | *I **agree**.* | *Agree* is not passive; incorrect form |
| *The meeting will be being held.* (most contexts) | *The meeting will be held.* | Future continuous passive is rare; avoid |
| *She explained me the problem.* (passive attempt) | *The problem was explained **to me**.* | *Explain* cannot take indirect object as subject |
